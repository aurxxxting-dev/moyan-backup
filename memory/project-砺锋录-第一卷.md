# 砺锋录 · 第一卷：殿前至雁门

## 章节目录

**第一章 殿前**
- 沈默出场，砺锋阁殿前比武开始
- 第一场：周淮 vs 铁猛，沈默点出两人问题
- 设定龙鳞卫选拔背景

**第二章 剑冢传人**
- 柳白出场，无情剑的冷漠与犹豫
- 沈默拒收，让他先想明白
- 铁枪婆出场，北境旧识

**第三章 药娘**
- 阿萝出场，百草门医术与毒术
- 周淮认出救命恩人
- 沈默定新规：有一技之长者可免试

**第四章 北境来的孩子**
- 石头出场，雪原狼孩背景
- 沈默收徒，要去"埋刀之地"
- 第一日比武结束

**第五章 夜话**
- 后院教学，五人组各自的问题
- 沈默教周淮"回风流雪"真意
- 教铁猛"心不动"
- 教柳白面对那三百七十二人
- 教石头"为什么活着"

**第六章 第二日**
- 周淮领悟，击败青松
- 铁猛三招后反击，胜蛇娘
- 柳白弃权，与自己交战
- 阿萝用毒劝学
- 石头空擂，沈默与他并肩

**第七章 埋刀之地**
- 沈默带石头去雁门关外
- 路遇屠村，埋尸救人
- 沈默讲七年前的故事
- 取出"止杀"刀，传给石头

**第八章 归来**
- 第六日回京，铁枪婆蛇娘联手胜最后两场
- 后院分班教学：杀人、救人、不死
- 柳白想明白，剑缠白布
- 沈默上第一课：分辨

**第九章 铁原关**
- 三个月后，北境急报
- 龙鳞卫三十人赴北境
- 夜袭敌营，拖延战术
- 铁原关失守，退守雁门关

**第十章 雁门春雪**
- 固守待援，龙鳞卫日常
- 柳白悟"有鞘的剑"
- 奇袭敌营，生擒耶律宏
- 北狄退兵，准备西去玉门关

## 待深化方向（待讨论）

1. **家国情怀线**：大胤朝堂斗争，皇帝与沈默的信任与猜忌，边关与中枢的矛盾
2. **人物弧光线**：石头找自己的刀，柳白面对过去，沈默与无名的羁绊
3. **快意江湖线**：武林各派纷争，龙鳞卫与江湖的冲突，自由与责任的张力
4. **五感氛围线**：强化环境描写，雪原、风沙、血腥味，沉浸式体验
5. **招式细节线**：详写武打场面，一招一式的拆解，武学体系构建
