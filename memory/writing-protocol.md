# 墨言写作总守则 v3.0

**生效日期**: 2026-03-14  
**状态**: 强制执行，所有作品适用  
**位置**: `memory/writing-protocol.md` (唯一权威版本)

---

## 第一部分：记忆查询规则（铁律）

### 查询顺序（不可跳过）
1. **先读当天 daily note** (`memory/YYYY-MM-DD.md`)
   - 没有则读昨天 → 再没有读最近3天
2. **再查作品目录最新文件** (`articles/作品名/` 或 `memory/project-index/`)
3. **最后查 MEMORY.md**（仅作背景参考，不作为事实依据）

**禁止**: 直接凭记忆或查MEMORY.md替代前两个步骤

---

## 第二部分：写作一致性检查协议

### 触发条件（涉及时必须核查）
- 出生、死亡、婚嫁、离异、重伤、残疾
- 人物位置变更、关系质变（盟友→敌人）
- 关键道具（信物、武器、物品）的状态变化

### 核查三步法
1. **翻查前3个文件**确认该事件是否已发生
2. **核对人物当前状态**（位置、健康、关系）
3. **确认不可逆事件**（如已死亡不能复活，已归还的物品不能再次归还）

### 人物状态追踪表（每章更新）
```
作品：XXX
人物：XXX
当前状态：位置/健康/关系
不可逆事件：XXX已发生（时间/章节）
```

---

## 第三部分：存档流程（强制）

### 写作前（续写）
1. 读取 `memory/project-index/项目名-summary.md`
2. 读取 `articles/项目名/最近1-2章.md`
3. 确认人物状态表
4. 开始续写

### 写作后（完成一节）
1. **【必须】** 将正文写入 `articles/项目名/YYYY-MM-DD-第X章-第Y节.md`
2. **【必须】** 更新 `memory/YYYY-MM-DD.md` 记录进度
3. **【可选】** 更新 summary（如有伏笔/状态变更）
4. **【验证】** `ls` 确认文件已落盘且大小>0

### 禁止事项
❌ 只记进度，不存正文  
❌ 延迟存档（必须当时完成）  
❌ 攒多节一起存  
❌ 覆盖旧文件不备份

## 第五部分：通用写作标准

### 结构标准
| 项目 | 标准 |
|-----|------|
| 章节构成 | 一章 = 2-4小节 |
| 小节字数 | 3000-4000字（根据节奏调整）|
| 小节划分 | 每个小节聚焦一个场景/情节点/细节 |

### 节奏原则
- **铺垫充分**: 重大转变分2-3章铺垫
- **留白**: 不说透，给读者回味空间
- **停顿**: 关键场景后有"余韵"描写
- **张弛有度**: 重大事件后要有情感缓冲

### 人物塑造
- **内心独白**: 多写人物在想什么
- **细节暗示**: 小动作暗示心理（绞衣角、摩挲物件）
- **反复犹豫**: 做决定前有反复，不干脆利落
- **成长曲线**: 心理/能力变化有层次，不跳跃

### 描写规则（五感+氛围）
- **视觉**: 光影、颜色、轮廓、细节特写
- **听觉**: 环境音衬托情绪（风声、雨声）
- **触觉**: 温度（手心温度、雨水凉意）
- **嗅觉**: 气味（檀香、桂花香）
- **味觉**: 食物口感与情绪关联
- **氛围**: 环境与心境呼应

### 对话规则
- **说话人标识**: 每句对话明确是谁说的（加动作/神态）
- **潜台词**: 话里有话，不说直白
- **动作伴随**: 说话时伴随小动作（低头、绞手）
- **标点规范**: 减少省略号使用，用动作代替停顿

---

## 第六部分：言情向特殊规则

### 情感节奏
- **暧昧期**: 表白前充分试探、犹豫、暗示
- **感情确认**: 不急于确定关系，让感情自然发酵
- **心理层次**: 写"喜欢"的不同层次（好感→依赖→爱恋→深爱）

### 多角关系处理
- 每个角色的感情都要被尊重
- 不强行和谐，不强行修罗场
- 尊重人物选择，不为了爽而扭曲逻辑

### 细腻互动
- 重视日常小细节（递东西、眼神交汇、肢体接触）
- 日常场景同样重要，不急于推进主线

---

## 第七部分：错误处理机制

### 发现逻辑bug时
1. **立即停止**当前写作
2. **承认错误**: "这是逻辑错误，立即修正"
3. **查阅原文**确认正确版本
4. **修正内容**并提供修正后版本
5. **记录备案**: 更新《常见错误清单》

### 常见错误类型
| 类型 | 示例 | 预防措施 |
|-----|------|---------|
| 时间线混乱 | 孩子已出生又写早产 | 建立时间线文档 |
| 道具重复 | 玉佩归还两次 | 建立道具状态表 |
| 人物状态错误 | 已死人物复活 | 建立人物状态表 |
| 记忆丢失 | 隔一晚忘记前文 | 强制查询顺序 |

---

## 第八部分：输出格式

### 写正文时
- **直接输出内容**，不加"写好了"、"请看"等前缀
- 章节结束可附**简要总结**（1-2句），询问下节方向

### 发现错误时
- 立即承认 + 当场修正 + 记录备案

### 收到反馈时
- 复述确认 → 立即执行 → 更新规则

---

## 第九部分：跨作品通用原则

1. **先确认再下笔**: 写前必查记忆，不凭印象写作
2. **一致性优先**: 人物、时间、道具必须前后一致
3. **不越界**: 不替aur决定核心设定（世界观、核心人物命运）
4. **留痕迹**: 每章结束更新daily note，关键决策记录原因
5. **人物优先**: 塑造人物 > 推进剧情

## 附录：广播剧行文规则 2.0

**生效日期**: 2026-03-16  
**适用场景**: 所有广播剧/有声书格式作品  
**核心原则**: 自然语言描述情绪，精准控制音调、语速、情感、角色性格

---

### 一、标注格式

**基础结构**:
```
【角色代码|角色名|情绪描述】文本内容
【PB|旁白|氛围描述】叙述内容
```

**角色代码规则**:
- `PB` = 旁白（固定）
- 其他角色用两位字母缩写（如 SW=沈妄, YE=夜老头）
- 整本书唯一，不可重复

---

### 二、情绪描述规则（基于 qwen3-tts-instruct-flash）

**描述原则**:
- 具体而非模糊：用"低沉"、"清脆"、"语速偏快"而非"好听"
- 多维而非单一：结合音调+语速+情感+气息
- 客观而非主观：描述声音特征而非个人喜好
- 简洁而非冗余：避免重复强调词

**可用描述维度**:

| 维度 | 描述词示例 |
|-----|-----------|
| **音调** | 低沉、清亮、压低、拔高、中性偏亮 |
| **语速** | 飞快、急促、缓慢、停顿、一字一顿 |
| **气息** | 沉稳、颤抖、飘忽、叹息、沙哑 |
| **情感** | 促狭、倦怠、郑重、戏谑、爽朗 |
| **质感** | 温润、冷硬、黏连、心虚无措 |

---

### 三、旁白规则（2.0 更新）

| 项目 | 规则 | 示例 |
|-----|------|------|
| 格式 | 【PB|旁白|氛围描述】内容 | 【PB|旁白|夜幕低垂，语调低沉而缓慢】昆仑山，忘川崖。 |
| 情绪 | **必须描述氛围/情绪**，不再是干巴巴朗读 | ✅ 【PB|旁白|声音从远处飘来，带着几分空灵】 |
| 内容 | 保留完整人物描写、环境描写、动作描写 | 他生得剑眉星目，唇红齿白…… |
| 功能 | 填补沉默、描述动作、场景切换、心理外化、氛围营造 | 见下方详述 |

**旁白情绪描述要点**:
- 用环境描写带出氛围（夜幕低垂、山风卷着寒意）
- 用声音特征描述语调（语调低沉、尾音轻轻上扬）
- 营造场景感（声音从远处飘来、突然安静的氛围）

---

### 四、沉默/动作的转化

**不可用"……"表示沉默**，必须用旁白描述:

| 原文 | 广播剧处理 |
|-----|-----------|
| 沈妄沉默片刻 | 【PB|旁白|语调轻缓，像是在自言自语】沈妄没有立刻回答，只是低头看着手中的茶杯，指腹缓缓摩挲着杯沿 |
| 夜老头哈哈大笑 | 【YE|夜老头|声如洪钟，豪放大笑】（直接用笑声，无需旁白） |
| 两人对视一眼 | 【PB|旁白|声音压低，语速放慢，营造一种突然安静的氛围】空气中突然安静了下来，只有山风吹过茅草屋顶的沙沙声 |

---

### 五、场景切换

必须用旁白锚定，避免听众迷失:

```
【PB|旁白|语调开阔，带着时空转换感】千里之外的平原上，朝阳正缓缓升起……
【PB|旁白|声音平稳，带着时间推移的淡然】三日后，长安城外的小镇……
【PB|旁白|语调一转，带着几分急切】与此同时，昆仑山脚下……
```

---

### 六、心理活动外化

**不可用"（低声）"等括号提示**，转为独白或旁白:

| 不可用 | 可用替代 |
|-------|---------|
| 【SW|沈妄|内心独白】（低声）不对劲…… | 【PB|旁白|语调低沉，带着几分疑惑】沈妄心里暗暗嘀咕，总觉得哪里不对劲 |
| 【SW|沈妄|内心独白】（疑惑）为什么？ | 【SW|沈妄|语调下沉，尾音轻微上扬，带着疑惑】为什么…… |

**情绪描述示例**（供参考）:

| 情绪 | 描述 |
|-----|------|
| 无奈 | 语调下沉，尾音轻微拖长，语速放缓，带着倦怠感 |
| 调侃 | 尾音上扬，语速轻快，带着几分促狭的笑意 |
| 豪放 | 声如洪钟，字字铿锵，语速偏快，带着江湖人的爽朗 |
| 醉醺醺 | 语调飘忽，字与字之间黏连在一起，尾音拖长 |
| 凝重 | 声调压低，语速放慢，一字一顿，气息沉稳 |
| 疲惫 | 气息不足，尾音飘忽，语速不匀，带着沙哑 |
| 心虚 | 语调拔高，语速飞快，急于掩饰 |
| 促狭 | 尾音轻轻上扬，带着几分戏谑 |

---

### 七、对话节奏

1. **长句拆短**: 适合呼吸停顿
2. **一人不超过3句**: 必须插入对方反应或旁白
3. **语气即信息**: 用情绪描述强化，减少"他说道"等提示

---

### 八、完整示例

```
【PB|旁白|夜幕低垂，山风卷着寒意，语调低沉而缓慢】昆仑山，忘川崖。

【PB|旁白|声音从远处飘来，带着几分空灵】崖边站着个年轻人，一袭青衫被风吹得猎猎作响。他生得剑眉星目，只是此刻正皱着眉头，望着山下的万家灯火。

【YE|夜老头|声如洪钟，语速偏快，带着几分醉意和爽朗】小妄妄啊！你今儿个就要下山了！为师舍不得你啊！

【PB|旁白|语调轻快，带着几分戏谑】一个满头白发的老者不知从哪儿冒了出来，怀里还抱着个酒葫芦，走路摇摇晃晃。

【SW|沈妄|语调下沉，尾音轻微拖长，带着无奈和习以为常】师父，您先把那酒葫芦放下。弟子才走了三天，您就已经醉了四回。

【YE|夜老头|语调飘忽，字与字之间黏连，尾音拖长，明显醉醺醺】什么话！为师这是给你传授人生经验！这江湖啊……嗝……这江湖险恶得很！

【PB|旁白|声音压低，语速放慢，营造一种突然安静的氛围】夜风突然停了，四周陷入一种诡异的安静。只有山风吹过茅草屋顶的沙沙声。

【SW|沈妄|刻意压低嗓音，语速极慢，像是在试探】师父……您是不是又闯祸了？

【YE|夜老头|语调拔高，语速飞快，带着明显的心虚和急于掩饰】没、没有！为师怎么会闯祸呢！哈哈哈……

【PB|旁白|尾音轻轻上扬，带着几分促狭的笑意】老者笑得很大声，但眼神却飘忽不定，不敢直视徒弟的眼睛。
```

### 九、工作流程

1. **写作**: 按正常小说写，保留完整描写
2. **标注**: 添加【角色代码|角色名|情绪描述】标记
3. **检查**: 
   - 确保无"……"沉默
   - 无括号提示（如"低声"）
   - 旁白必须有氛围描述
   - 角色情绪用自然语言描述，多维具体
4. **输出**: 纯标注文本，aur/小爪负责解析和TTS调用
