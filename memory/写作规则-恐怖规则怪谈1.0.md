# 写作规则扩展 - 恐怖/规则怪谈 1.0

> 基于基础规则1.0，针对恐怖、悬疑、规则怪谈类小说的扩展

---

## 一、恐怖氛围构建规则

### 1.1 恐惧层次（由浅入深）

| 层次 | 表现方式 | 示例 |
|------|----------|------|
| **不安** | 细微异常（温度变化、声音不对） | 走廊的灯明明没坏，却一直在闪 |
| **紧张** | 明确威胁但未显现 | 知道有东西在追，但还没看到 |
| **恐惧** | 直面恐怖元素 | 看到怪物的真面目 |
| **绝望** | 无路可逃、规则无法破解 | 发现出口被封死，规则自相矛盾 |

### 1.2 恐怖元素使用原则

| 原则 | 说明 |
|------|------|
| **少即是多** | 怪物/异常不要频繁出现，保持神秘 |
| **延迟揭示** | 真相分阶段揭露，不一次给完 |
| **日常异化** | 熟悉场景出现微小异常（镜子里的自己慢了半拍） |
| **无法言说的恐惧** | 有些恐怖无法用语言描述，用人物反应暗示 |

## 二、规则怪谈特殊规则

### 2.1 规则设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **表面合理** | 规则看起来像是为了保护 | "请勿在凌晨2点照镜子" |
| **隐藏矛盾** | 规则之间存在矛盾，需要推理 | 规则A说"红灯时不能动"，规则B说"看到红灯必须马上跑" |
| **违反后果** | 违反规则的后果要延迟显现 | 违反后当时没事，几小时后才开始异变 |
| **渐进陷阱** | 前期规则安全，后期规则开始坑人 | 前3条规则是对的，第4条开始是陷阱 |

### 2.2 规则呈现方式

| 方式 | 效果 |
|------|------|
| **官方通知** | 广播、告示牌、手册（冰冷正式） |
| **前人留言** | 涂鸦、笔记、血字（混乱恐惧） |
| **试错总结** | 主角自己摸索（读者共鸣） |
| **自相矛盾** | 不同来源的规则冲突（制造焦虑） |

## 三、悬疑/解谜节奏规则

### 3.1 信息释放节奏

| 阶段 | 信息释放量 | 读者状态 |
|------|-----------|----------|
| **铺垫期** | 20% - 抛出异常现象 | 好奇 |
| **探索期** | 40% - 发现规则、收集线索 | 投入解谜 |
| **危机期** | 30% - 规则矛盾、真相浮现 | 紧张焦虑 |
| **揭露期** | 10% - 真相大白 | 恍然大悟/震惊 |

### 3.2 伏笔与回收

| 类型 | 做法 |
|------|------|
| **视觉伏笔** | 早期提到的细节，后期成为关键线索 |
| **规则伏笔** | 某条看似无关的规则，后期救命 |
| **人物伏笔** | 某个角色的异常，后期揭示原因 |
| **环境伏笔** | 场景的某个特征，后期成为破解点 |

## 四、逃离类结构规则

### 4.1 场景设计

| 场景 | 功能 | 恐怖元素 |
|------|------|----------|
| **安全区** | 短暂休整、整理线索 | 逐渐缩小或被侵蚀 |
| **探索区** | 收集信息、触发事件 | 规则适用、异常频发 |
| **危险区** | 必死/高代价区域 | 怪物出没、规则失效 |
| **出口** | 最终目标 | 需要满足特定条件才能开启 |

### 4.2 逃离进程设计

| 阶段 | 地图开放度 | 难度 |
|------|-----------|------|
| **被困** | 小范围（教室/宿舍） | 规则简单 |
| **探索** | 中等（整层楼） | 规则复杂、出现矛盾 |
| **深入** | 大范围（整栋楼/全校） | 规则陷阱、怪物追击 |
| **突围** | 出口路径 | 最终考验 |

## 五、人物塑造（恐怖向）

### 5.1 主角特质

| 特质 | 作用 |
|------|------|
| **理性** | 分析规则、保持冷静 |
| **观察力** | 发现细节、察觉异常 |
| **牺牲精神** | 必要时为他人冒险 |
| **成长** | 从恐惧到勇敢，从盲目到智慧 |

### 5.2 配角功能

| 类型 | 作用 | 结局倾向 |
|------|------|----------|
| **向导型** | 解释规则、提供线索 | 可能牺牲 |
| **怀疑型** | 质疑规则、制造冲突 | 可能因不信规则而死 |
| **恐慌型** | 制造紧张、触发危机 | 早死警告 |
| **隐藏型** | 身份成谜、可能是幕后黑手 | 反转角色 |

## 六、特殊描写技巧

### 6.1 恐怖场景描写

| 技巧 | 示例 |
|------|------|
| **延迟描写** | 先写人物反应，再写看到什么 |
| **碎片信息** | 通过镜子反光、余光等间接呈现 |
| **感官剥夺** | 黑暗、静音，让读者和主角一样看不见/听不到 |
| **认知失调** | 熟悉的场景出现不可能的几何结构 |

### 6.2 规则呈现描写

| 技巧 | 示例 |
|------|------|
| **冰冷正式** | "本校规定：晚自习后请勿在走廊逗留" |
| **破损血迹** | "不...要...相...信...红...色..." |
| **矛盾并存** | 同一张纸上，两条规则互相否定 |
| **逐渐变异** | 再次查看时，规则内容变了 |

## 七、禁忌与注意事项

| ❌ 禁忌 | ✅ 正确做法 |
|--------|------------|
| 恐怖元素堆砌 | 留白，让读者想象 |
| 规则一次性给完 | 分批次、带陷阱、有矛盾 |
| 主角全程开挂 | 会犯错、会受伤、会恐惧 |
| 为吓人而吓人 | 恐怖服务于剧情和人物 |
| 逻辑漏洞 | 规则要有内在逻辑 |
