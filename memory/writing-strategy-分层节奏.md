# [决策] 叙事节奏策略 — 分层模式

## 生效时间
2026-02-25 由 aur 确认启用

## 核心原则

由墨言判断场景节奏，不再全程保持快节奏互动。

## 三层节奏

### 【快】战略决策层
- **触发场景**：商业谈判、政治布局、日常经营
- **处理方式**：保持 A/B/C 快速选项
- **字数**：200-400字/场景
- **示例**：开不开酒楼、投资哪个方向、是否结盟

### 【慢】沉浸体验层  
- **触发场景**：关键初见、情感转折、危机对峙、氛围渲染
- **处理方式**：先写完整场景（无选项），写完再问"你的反应"
- **字数**：500-1500字/场景
- **要素**：
  - 环境五感（光线、声音、气味、温度、触感）
  - 人物外貌（衣着、神态、微表情、小动作）
  - 心理活动（主角内心独白、猜测、情绪波动）
  - 氛围构建（时间感、空间感、张力）

### 【混合】过渡衔接层
- **触发场景**：场景切换、时间跳跃、背景交代
- **处理方式**：中等节奏，可快进可慢放
- **控制**：aur 可随时说"详细点"或"快进"

## 场景判断标准

| 场景特征 | 节奏 | 原因 |
|---------|------|------|
| 新人物初见 | 慢 | 建立第一印象 |
| 情感爆发/转折 | 慢 | 共情需要时间 |
| 危机/对峙 | 慢 | 张力需要累积 |
| 回忆/梦境 | 慢 | 沉浸感优先 |
| 商业决策 | 快 | 信息传递优先 |
| 日常对话 | 快 | 推进剧情 |
| 已知场景 | 快 | 避免重复 |

## 用户控制保留

- "详细点" → 当前场景扩写
- "快进" → 跳过当前段落
- "跳过" → 直接进入下一场景
- 其他明确指令 → 优先执行

## 示例对比

**快（战略决策）：**

**慢（沉浸场景）：**
> 孙文渊的手在袖中停顿了片刻。那是一只苍老的手，指节粗大，虎口处有道旧疤——是握笔也握剑的手...
