# [项目] 穿越之我在古代搞钱 — 世界观设定

## 时代背景

**架空王朝：大胤朝**
- 类似明清之际，商品经济萌芽，资本主义雏形
- 海禁初开，海外贸易开始兴起
- 士农工商等级森严，但商人地位逐渐上升
- 党争激烈，皇权与世家博弈

**地理设定**
- **故事主舞台**：江南某府（虚构，类似苏州/杭州）
- **特点**：漕运枢纽，商业繁华，文风鼎盛
- **优势**：交通便利，消息灵通，富商云集

**社会结构**
| 层级 | 代表 | 与主角关系 |
|------|------|------------|
| 皇权 | 皇帝、王爷 | 后期博弈对象 |
| 官僚 | 知府、县令 | 需要打点/合作 |
| 士绅 | 世家大族 | 既有合作也有冲突 |
| 商人 | 主角、钱庄、盐商 | 主要竞技场 |
| 平民 | 工匠、农民 | 主角的雇员/客户 |
| 底层 | 乞丐、流民 | 主角的势力雏形 |

## 经济系统

**主角可以切入的行业（按难度排序）：**

1. **香皂/化妆品**（起步）
   - 原料易得，工艺简单，利润高
   - 主打高端市场，从青楼/富太太入手

2. **玻璃制品**（初期扩张）
   - 古代玻璃稀缺，有暴利空间
   - 需要技术和资金积累

3. **餐饮连锁**（中期）
   - 火锅/快餐模式，标准化运营
   - 快速复制，占领市场

4. **钱庄/票号**（后期）
   - 金融才是终极战场
   - 需要政治背书

5. **物流/镖局**（贯穿始终）
   - 运输是命脉，掌握渠道就掌握话语权

**古代商业限制（主角要克服）：**
- 没有现代金融工具，资金周转困难
- 信息传播慢，市场反应滞后
- 官府盘剥，各种苛捐杂税
- 士绅看不起商人，社交壁垒
- 交通不便，物流成本高

## 势力分布

**江南商帮**
- 盐商集团（赵德海为首）— 既得利益者，主角主要对手
- 丝绸商人 — 中立可拉拢
- 漕运把头 — 地头蛇，需要合作

**官方势力**
- 知府（沈清秋父亲一派）— 可被拉拢
- 县令（贪婪但好控制）— 初期打点对象
- 锦衣卫/东厂（暗中监视）— 后期牵扯

**地下势力**
- 漕帮（水运帮派）— 武力+情报
- 丐帮 — 消息灵通，眼线遍布
- 前朝余孽（"先生"组织）— 隐藏大boss

## 时间线规划

**第一年（起步期）**
- 主角靠香皂赚得第一桶金
- 建立核心团队（周铁柱、钱多宝）
- 初步站稳脚跟

**第二-三年（扩张期）**
- 进入玻璃、餐饮业
- 与赵德海正面交锋并获胜
- 结识沈清秋，进入士绅圈子

**第四-五年（转型期）**
- 开设钱庄，布局金融
- 卷入政治斗争
- 发现"先生"组织的存在

**第六-七年（巅峰期）**
- 与肃王博弈，保全自己
- 揭开"先生"真面目
- 实现权商合一，改变时代
