# 项目：《逃离诡异校园》- 规则怪谈

## 核心设定

| 项目 | 内容 |
|------|------|
| **类型** | 恐怖 / 规则怪谈 / 克苏鲁元素 |
| **场景** | 封闭校园（宿舍→教学楼→全校） |
| **主角** | 普通大学生，宿舍醒来发现异常 |
| **规则来源** | 手机不知名群聊（逐渐显现的规则） |
| **恐怖元素** | 超自然 + 克苏鲁意味（认知污染、不可名状） |
| **结局** | 发现更大阴谋（系列开篇） |

## 故事框架

### 开端（被困）
- 主角在宿舍醒来，发现室友消失
- 手机收到不知名群聊消息
- 第一条规则出现

### 发展（探索）
- 验证规则真假
- 遇到其他被困学生
- 规则逐渐增多，出现矛盾

### 高潮（真相）
- 发现规则背后的逻辑
- 真假反转
- 某个"同伴"的真实身份

### 结局（阴谋）
- 逃出生天但发现更大的局
- 留下悬念

## 人物设定

| 角色 | 身份 | 功能 | 反转可能 |
|------|------|------|----------|
| **主角（待定名）** | 普通学生 | 推理、成长 | 无，读者视角 |
| **室友A** | 醒来时不在 | 早期失踪，后期出现 | 可能已被"替代" |
| **神秘群友"管理员"** | 发布规则 | 引导/误导 | 是敌是友？ |
| **另一位幸存者** | 被困学生 | 互动、合作/背叛 | 真实身份？ |

## 规则示例（待细化）

1. **基础规则**
   - 如果听到走廊有脚步声，请立即关灯并保持安静
   - 镜子里的你如果眨眼了，那不是你的倒影

2. **进阶规则（带矛盾）**
   - 规则A：红灯亮起时必须留在原地
   - 规则B：看到红灯请立即逃离

3. **克苏鲁元素规则**
   - 如果你开始理解规则背后的逻辑，请立即停止思考
   - 不要试图看清"它"的全貌

## 恐怖节奏设计

| 阶段 | 恐怖类型 | 规则复杂度 |
|------|----------|-----------|
| 宿舍阶段 | 不安（细微异常） | 简单规则 |
| 走廊探索 | 紧张（未知威胁） | 规则增多 |
| 教学楼 | 恐惧（直面异常） | 规则矛盾 |
| 真相揭露 | 绝望+认知污染 | 规则背后的真相 |

## 核心卖点

1. **规则解谜** - 读者和主角一起推理
2. **真假反转** - 信任与背叛
3. **克苏鲁氛围** - 不可名状的恐惧
4. **校园代入感** - 熟悉场景的异化
