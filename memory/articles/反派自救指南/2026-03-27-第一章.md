# 《反派自救指南》

## 第一章：这三百个孩子，我不要

陈渊睁开眼的时候，面前跪着三百个童男童女。

他们穿着统一的素白袍子，被符咒定在地上，不哭不闹，眼神空洞。祭坛周围插着七十二面玄阴幡，无风自动，发出婴儿啼哭般的声响。

陈渊花了三秒钟确认三件事：

第一，他穿越了。

第二，他穿成了《九霄仙途》里的终极大反派，玄阴宗宗主万魂真人。

第三，原书第一章，万魂真人用这三百个童男童女祭炼本命法宝"噬魂幡"，从此坐实魔头之名，被主角团记在小本本上，三百万字后轰杀至渣。

【叮——反派扮演系统激活】

一个欢快的声音在他脑子里响起，像是超市促销的广播。

【宿主：陈渊（万魂真人）】
【当前剧情点：血祭童魂（原著第一章）】
【系统任务：维持万魂魔头人设，完成血祭】
【任务奖励：寿元+1年】
【失败惩罚：神魂俱灭（即死）】
【倒计时：00:09:59】

陈渊看着那三百双空洞的眼睛，又看了看倒计时。

九分钟。

他得在九分钟内，既救了这些孩子，又不崩人设，还要让在场所有人都相信他是个十恶不赦的大魔头。

"有点意思。"陈渊说。

他声音嘶哑，像是砂纸摩擦骨头——这是万魂真人的原声，自带恐怖混响效果，省得他自己配音了。

祭坛下方站着几个玄阴宗长老，为首的是个面色青灰的老者，道号"千尸老人"，正用一种"宗主您今天想先吃哪个"的期待眼神看着他。

陈渊清了清嗓子。

"本尊观这三百童魂……"他故意停顿，让声音在祭坛上回荡，"资质粗劣，不堪大用。"

千尸老人一愣："宗主，这……这可是按您吩咐，从七十二个凡人城镇精挑细选……"

"本尊说，"陈渊打断他，语气里带着那种"你居然敢质疑我"的愠怒，"他们，不，合，格。"

他站起来——这具身体修为高深，随便一动就带起阴风阵阵，衣袍翻飞如夜鸦展翅，视觉效果满分。

陈渊走到最近的一个童男面前。那孩子约莫七八岁，脸色苍白，瞳孔涣散，显然被下了禁制。陈渊伸手，捏住孩子的下巴，左右打量，像在挑猪肉。

"根骨三品，勉强能入外门。"他松开手，嫌弃地甩了甩袖子，"这种货色，也配进本尊的噬魂幡？"

千尸老人脸都绿了："宗主，那您的意思是……"

"扔了。"陈渊转身，背对着那三百个孩子，声音冷漠，"送给山下那些小门派，当奴仆也好，当药人也罢，随他们处置。本尊的噬魂幡，要的是极品阴魂，不是这种……"

他顿了顿，想了个词：

"……残次品。"

【叮——】

系统提示音响起。

【行为判定中……】

陈渊心跳漏了一拍。他在赌——赌系统只看"外界认知"，不看真实意图。在场所有人听到的都是：万魂真人嫌这批货质量太差，不要了，扔给别人当奴隶或药材。

从外界视角，这是标准的反派行为：冷酷、残忍、视人命如草芥。

但实际上，这些孩子从"必死"变成了"活着"——虽然当奴仆或药人也不怎么样，但至少还有逃命的机会，还有被救的可能。

总比魂飞魄散强。

【判定完成】
【外界认知：万魂真人残忍无道，因资质不足便弃如敝履】
【人设契合度：87%】
【任务完成度：血祭取消，替代方案符合反派逻辑】
【奖励发放：寿元+1年】

陈渊松了口气。

但他还没来得及高兴，系统又响了：

【新任务生成：七日后，主角叶辰将抵达玄阴宗地界，请宿主做好"接待"准备】
【任务要求：按原著给予叶辰必要的"磨难"，推动其成长】
【特别提示：叶辰乃天道之子，不可击杀，否则……您懂的】

陈渊："……"

他想起原著剧情：万魂真人随手一挥，把刚入门的主角打成重伤，主角掉下悬崖，获得奇遇，从此走上开挂之路。

所以他的任务是：当那个把主角打下悬崖的坏人。

而且不能真打死，还得控制好力度，让主角"恰好"掉下去，"恰好"没死，"恰好"获得传承。

这叫什么？

这叫技术活。

陈渊坐回白骨王座，看着千尸老人指挥弟子把那三百个孩子带走。那些孩子还是眼神空洞，但至少，他们还能看见明天的太阳。

"宗主，"千尸老人回来复命，谄媚地笑着，"那今日的血祭……"

"延后。"陈渊淡淡道，"本尊要闭关七日，参悟更高深的魂道。"

"是是是，宗主圣明！"

千尸老人退下了。

祭坛上只剩下陈渊一个人，和七十二面啼哭的玄阴幡。

他盯着那面还没炼成的噬魂幡，心想：原著里，这幡里最后收了十万生魂。现在第一笔生意就没做成，后面怎么办？

总不能真的去杀人吧？

陈渊叹了口气。

他想起穿越前看的那些"反派自救"小说，人家要么有系统商城可以买道具洗白，要么能接到正经的行善任务。

他的系统呢？

让他当魔头，还不能真当，得假装当，还得当得所有人都信。

这叫什么反派自救？

这叫反派演技进修班。

【叮——系统提示：宿主对本系统有误解】

"哦？"陈渊在心里问，"那你倒是说说，我该怎么自救？"

【本系统名为"反派扮演系统"，核心逻辑：只要外界认为您是反派，您就是反派】
【至于您内心是否善良、是否救人、是否拯救世界，系统不关心】
【建议宿主：在维持人设的前提下，做任何您想做的事】

陈渊愣了一下。

"你是说……"

【您可以把这三百个孩子当药人，也可以偷偷给他们下禁制保护；您可以轰杀主角，也可以把他打落悬崖让他获得奇遇】
【外界看到的是什么，才重要】
【温馨提示：宿主当前人设契合度87%，尚有13%的发挥空间】

陈渊沉默了。

他看着远处被押走的孩子们，又看了看倒计时归零后恢复平静的界面。

原来如此。

这不是要他当坏人。

这是要他……当一个所有人都以为是坏人的人。

而他可以在这个"坏人"的壳子里，做任何他想做的事。

陈渊忽然笑了。

这有点意思。

他站起身，玄阴幡在他身后猎猎作响，像是无数冤魂在哭嚎。他张开双臂，任由阴风灌满衣袍，对着空荡荡的祭坛，用一种能让方圆十里都听见的声音说道：

"本尊要闭关七日。七日之后……"

他顿了顿，想起那个叫叶辰的主角。

"……本尊要亲自，会会那个叫叶辰的小子。"

声音传出玄阴宗，传入山林，传入远处某个正在赶路的少年耳中。

那少年打了个寒颤，握紧手中的木剑，眼神更加坚定了。

而陈渊——万魂真人，玄阴宗宗主，世人口中的大魔头——正坐在白骨王座上，盘算着怎么在"轰杀"主角的时候，让他"恰好"掉下那个有奇遇的悬崖。

难度不小。

但总比真的杀人强。
