# 《第九次出发》第一章（续二）

周茹是被阳光晒醒的。

不是那种温柔的晒，是西北七月早上九点的暴晒，透过挡风玻璃直直地打在她脸上。她皱着眉睁开眼睛，一时间不知道自己在哪里。睡袋的拉链已经被她无意识中蹬开了半截，身上出了一层黏糊糊的汗。

她坐起来，脖子僵硬得像是锈住的轴承。

驾驶座是空的，许林不在。周茹心里紧了一下，但随即看见副驾座上的双肩包还在，那根系着包带的红塑料绳垂在车座上，像是一截断了的血管。

她推开睡袋，从后座爬到前面，眯着眼睛看向车外。

许林站在服务区的边缘，背对着车，正在打电话。她一只手举着手机，另一只手在空中比划着什么，动作很大，像是在争论。周茹看不见她的表情，但能看见她的肩膀绷得很紧。

周茹没急着出去。她坐在驾驶座上，从储物格里摸出一盒薄荷糖，倒了两颗放进嘴里。清凉的甜味冲淡了嘴里的苦涩，她慢慢地等那阵刚睡醒的眩晕过去。

许林打完了电话，站在原地没动。她低着头，肩膀慢慢垮下来，刚才那种紧绷的姿态消失了，取而代之的是一种周茹很熟悉的东西——那种被什么东西压过之后，暂时无力反抗的松弛。

周茹推开车门下车。

许林听见了声音，转过身。她的眼睛有点红，但脸上没什么表情，像是已经习惯了在被人看见之前把情绪收好。

"早。"她说。

"嗯。"周茹走过去，站在她旁边，两个人一起看着服务区外面那条伸向远方的公路。路面上已经有热气在扭曲，远处的地平线在晃动的空气里变得模糊不清。

"你电话打完了？"周茹问。

"打完了。"许林说，然后停顿了一下，像是在决定要不要继续说，"是我爸。"

周茹没接话。她等了一会儿，许林果然自己说了下去。

"他不知道我出来了。"她说，"我骗他说我在学校。"

"学校？"

"嗯。我骗他说我在读大专，其实早就不读了。"

周茹点点头，没评价。这种事她见得多了。孩子撒谎，父母有时候知道有时候不知道，但大多数时候是假装不知道。大家都累。

"他说什么？"周茹问。

"没说什么。"许林扯了扯嘴角，像是想笑但没笑出来，"就问我钱够不够。我说够。"

周茹从口袋里摸出那盒薄荷糖，递给许林。许林愣了一下，然后倒了一颗出来，放进嘴里。

"走吧。"周茹说，"还要赶路。"

她们回到车上，周茹发动引擎。车子抖了一下，发出一阵熟悉的轰鸣，然后平稳下来。周茹听着那声音，确认没有异常——这是她的习惯，每次启动都要听一听，像是医生听心跳。

"你吃早饭吗？"她问。

"不吃。"许林说，"我不饿。"

周茹也没吃。她不习惯在服务区的餐厅吃东西，那些东西贵，而且味道千篇一律。她宁愿忍到库尔勒，找个路边的小馆子，吃一碗真正的拉面。

车子驶出服务区，汇入车流。

今天的路和昨晚不一样。昨晚是黑沉沉的一团，只有车灯照亮的那么一小块。现在是敞亮的，能看见路的尽头连着天，天尽头有一些模糊的山影。周茹喜欢这种视野，它让人觉得目的地是可以到达的，哪怕实际上还远得很。

"还有多远？"许林问。

"三百多公里。"周茹说，"快的话，下午到。"

许林"嗯"了一声，没再说话。她靠在车窗上，看着外面掠过的风景。戈壁滩还是那片戈壁滩，但在阳光下呈现出不同的层次——近处的沙砾是灰黄的，远处的山是铁青的，再远就是一片白茫茫的雾气。

周茹开着车，偶尔看一眼许林。女孩的脸在车窗的反光里忽明忽暗，像是一张被水浸泡过的旧照片。

她们在一个路边的小加油站停下来加油。加油站很旧，只有两个油枪，旁边搭着一个棚子，卖一些矿泉水和方便面。周茹加油的时候，许林去棚子里买了一瓶水，回来递给周茹。

"给你。"她说。

周茹接过来看了一眼，是冰的，瓶壁上结着一层水珠。她拧开喝了一口，凉得牙疼，但精神一振。

"多少钱？"她问。

"三块。"许林说，"不用给我。"

周茹没坚持。她喝完水，把瓶子放在杯架上，然后发动车子继续上路。

那瓶水像是一个转折点。从那之后，车里的气氛变得松了一些。许林开始主动说话，讲一些无关紧要的事——她妈以前跑长途时遇到的趣事，她小时候跟着车去过的那些地方，她记得住的一些地名和完全不记得的另一些。

周茹听着，偶尔应一声。她不太会讲故事，但她会听。听那些琐碎的、没有目的的叙述，这是她这么多年跑长途学会的东西之一。

下午三点，她们看见了库尔勒的第一片建筑。

那是城郊的工业区，一些厂房和仓库，灰色的、方方正正的，在烈日下显得有些荒凉。周茹按照导航的指引，穿过工业区，进入老城区。

老城区的街道很窄，两边种着白杨树，树叶在头顶交织成一片浓密的绿荫。周茹放慢了车速，在狭窄的街道上小心穿行。她不太熟悉这里，导航时不时发出"前方掉头"的指令，她选择无视，凭着自己的方向感继续往前。

"建国路。"许林忽然说，声音有点紧，"是这条路吗？"

周茹看了一眼路牌，"不是。前面路口右转。"

她们在下一个路口右转，进入一条更窄的巷子。巷子里停满了车，周茹不得不把车速降到几乎和走路一样慢。她看着两边的门牌号，数字在递增——十二、十四、十六……

然后她们看见了十七号。

那是一栋老式的三层楼房，外墙的白灰已经斑驳，露出底下的红砖。门口种着一棵不知道什么树，叶子落了一地，没人扫。门口挂着一个褪色的牌子，上面模糊地写着几个字，周茹看不清。

周茹把车停在路边，熄火。

许林坐在那里没动。她盯着那栋楼，手指无意识地抠着那个破双肩包的带子。

"到了。"周茹说。

"嗯。"许林的声音很轻。

她没立刻下车。周茹也没催她。她们就这样坐在车里，听着引擎冷却时发出的轻微的"咔咔"声，看着那栋老旧的房子。

"谢谢你。"许林忽然说，"送我到这里。"

"不用谢。"周茹说，"顺路。"

许林转过头，看着周茹。她的眼睛很亮，但不是要哭的那种亮，是一种下定决心的亮。

"我应该给你多少钱？"她问。

周茹想了想，"你有多少？"

许林从口袋里掏出一叠皱巴巴的钞票，数了数，"一百四十七。"

"给一百吧。"周茹说。

许林抽出一张一百的，递给周茹。周茹接过来，塞进仪表盘下面的储物格里。她没数，也没看真假。

许林推开车门，拎着那个用红塑料绳系着的双肩包，站在路边。阳光穿过树叶的缝隙，在她身上投下斑驳的影子。

"大姐。"她说，"你叫什么名字？"

"周茹。"周茹说，"周围的周，草字头下面一个如。"

"周茹。"许林重复了一遍，像是在记住这个发音。然后她说，"我叫许林，许诺的许，树林的林。"

"我知道。"周茹说，"你昨晚说过了。"

许林愣了一下，然后笑了。那是周茹第一次看见她笑，虽然笑得很浅，但确实是笑。

"那我走了。"许林说。

"嗯。"周茹说，"祝你好运。"

许林转身，走向那栋三层楼房。她的背影在斑驳的树影里越来越小，最后消失在门口的阴影里。

周茹坐在车里，看着她消失的方向，坐了很久。

她应该走了。她还有货要卸，还有回程的路要赶。但她没动。她掏出那盒薄荷糖，倒了一颗放进嘴里，慢慢地等那股清凉的味道在舌头上化开。

她想起许林说的那个地址，建国路十七号。她想起自己钱包里那张字迹模糊的纸条。她想起自己已经三年没有打开过那个钱包的夹层。

也许她应该进去看看。也许她应该至少知道，许林找到的那个人是谁。

但她没有。

她发动车子，调转车头，驶向城区的另一边。那里有她的货，有她的生活，有她必须继续开下去的路。

至于那个走进去的人，那是另一个故事了。

**第一章 完**
