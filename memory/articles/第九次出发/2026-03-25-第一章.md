# 《第九次出发》第一章

周茹把车停在服务区的时候，天边还剩一点亮。不是那种干净的亮，是西北特有的、混着沙土的昏黄，像有人把灯罩子裹了一层纱布。

她下车，从副驾驶座底下摸出暖水壶。壶是去年在兰州买的，不锈钢壳子凹进去一块，不影响用。她拧开盖子，倒了一盖水，蹲在轮胎旁边慢慢喝。水已经不太热了，温吞吞的，有一股铁锈味。她喝得很慢，一口，停一停，看着高速路上偶尔开过去的车灯。

这是她的第八趟。

八趟，从成都到喀什，再从喀什放空回来。她记得数字，但不记得日期。对她来说，时间不是日历上那一格一格的东西，是油箱里的油、轮胎的磨损、腰间盘突出的程度。上一次发作是在嘉峪关附近，她硬撑到哈密才找诊所。医生是个回民老头，不会说汉话，比划着让她趴下，给她拔了三罐。她趴在那里，脸埋进枕头里，闻着一股陈年棉絮和红花油混在一起的气味，忽然想起自己年轻时也这样趴过，那时候是为了别的——为了考文工团，压腿，疼得眼泪往枕头里渗。

那都多少年前的事了。

她把暖水壶盖拧回去，站起身，腰有点僵。她活动了一下，绕到车后，检查绑货物的绳子。其实不用检查，中午在张掖休息的时候刚查过。但她总得找点事做，不然站着也是站着。

服务区里没什么人。一个开大货车的男人从厕所出来，看了她一眼，没打招呼。她也不打。这条路上跑车的女人少，但不是没有。她不需要谁的特别关注，也不需要证明什么。五十二岁了，她早就过了需要证明什么的年纪。

她回到车上，把驾驶座的靠背往后调了调，打算眯一会儿。出发前她算过，按现在的速度，明天傍晚能到库尔勒。但那只是理论上的。路上会发生什么，她从来不算。算不准的。

她刚闭上眼睛，有人敲车窗。

不是服务区那种轻轻的叩，是有点急、有点重的三下。她坐起来，把窗户摇下去一条缝。

外面站着个女孩，二十出头，穿着件洗得发白的牛仔外套，背着一个双肩包，包带断了，用一根红色的塑料绳系着。女孩的脸被风吹得有点红，头发乱糟糟的，像是被车过的高速路带起来的风刮的。

"大姐，能搭个车吗？"

周茹没说话，看着她。女孩的手里攥着一张皱巴巴的纸，像是一张车票，又像是一张地图。她的指甲剪得很短，指节处有一道新鲜的划痕，血已经干了，结成一条暗红的线。

"去哪？"周茹问。

"库尔勒。"女孩说，然后像是怕被拒绝，又补了一句，"我可以给钱。"

周茹没说要，也没说不要。她沉默了几秒，那几秒里她听见服务区广播在播什么，听不清内容，只有电流的沙沙声。然后她伸出手，从里面把副驾驶的门锁打开了。

女孩愣了一下，像是没想到这么容易。她拉开车门，先把包扔进来，然后才坐上来。她身上有一股味道，不是香水，是肥皂和阳光晒过的棉布的混合，还有一种更底下的、像是久未洗澡的油脂味。不臭，但重。

"系安全带。"周茹说。

女孩手忙脚乱地找安全带，扣了三次才扣上。周茹发动车子，从服务区并入主路。天已经黑透了，车灯打出去，照亮前面不到三十米的路面，再往前就是一片混沌。

"我叫许林。"女孩说，声音有点哑，像是很久没喝水了。

周茹"嗯"了一声，没说自己叫什么。女孩等了一会儿，没等到回应，也就不说话了。

车里安静下来。只有发动机的声音，和轮胎碾过路面接缝时规律的"咯噔"声。周茹喜欢这种声音，它有节奏，像某种催眠的白噪音。她开了十二年的车，这种声音陪她走过了大部分醒着的时间。

许林坐在旁边，一开始坐得很直，双手抓着膝盖。慢慢地，她开始松下来，头歪向窗户那边，大概是想睡觉。但她没睡着，周茹能感觉到——她的呼吸不规律，一会儿快，一会儿慢，像是心里有事。

开了大概一个小时，周茹在一个路边的停车带停下来。这里不是正规的服务区，只是一个拓宽出来的应急车道，边上有一块牌子写着"前方连续下坡"。她熄火，下车，绕到车后撒尿。女孩没下车，坐在车里没动。周茹撒完尿，站在车后面点了一根烟。烟是红塔山，软包的，她抽了二十年了。

她抽完半根，才回到车上。

"你不怕我？"她忽然问。

许林转过头，像是被这个问题惊到了。车里没开灯，只有仪表盘的一点微光，映着女孩的脸，忽明忽暗。

"怕什么？"她问。

"怕我是坏人。"周茹说，语气平淡，不是威胁，只是陈述，"一个女的，半夜搭上一个陌生女人的车。你不怕？"

许林沉默了一会儿，然后说："我妈以前也是开大车的。"

周茹没说话，等她继续说。

"她跑了几年，后来不跑了。"许林的声音低下去，"嫁给了一个开修车铺的。"

"然后？"

"然后死了。"许林说，"三年前。病死的。"

周茹把烟掐了，发动车子。引擎的声音盖过了许林最后那句。她不确定自己是没听清，还是听清了但不想回应。她挂上档，车子重新驶入黑暗。

"你像你妈？"周茹问，眼睛看着前方。

"不像。"许林说，"她话多，我话少。"

"那你为什么跟我说这些。"

许林没回答。周茹也没再追问。路还长，她们有的是时间。或者没有。谁知道呢。

车窗外，戈壁滩在夜色里变成一片模糊的黑。偶尔有对面来车的灯光闪过，照亮许林的脸，又让她重新沉入黑暗。她的眼睛睁着，看着前方，但周茹知道她没有在看路。她在看别的什么东西，也许是记忆，也许是想象，也许只是纯粹的空白。

周茹曾经也有那种眼神。现在没有了。现在她只看路，看油表，看后视镜。那些别的，看太多没用。

她们就这样开下去。
