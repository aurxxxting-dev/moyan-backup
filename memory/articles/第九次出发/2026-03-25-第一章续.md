# 《第九次出发》第一章（续）

车开了很久，久到许林真的睡着了。

她的头歪在车窗上，随着车身的轻微颠簸一点一点。周茹从余光里能看见她，睡得很沉，眉头却还皱着，像是在梦里还在赶路。周茹没有叫醒她，也没有调低音乐——车里根本没有音乐，只有发动机和风声。

她看了眼仪表盘上的时间，凌晨两点十七分。按照计划，她应该在下一个服务区停下来，睡到天亮再走。但她忽然不想停了。停下来就要面对服务区里的灯光、其他司机的眼光、还有那种躺在床上却睡不着的感觉。不如一直开，开到身体逼她停下来为止。

她曾经是能熬夜的。年轻时跑长途，连着开十六个小时，到了地方卸完货还能去街边喝碗羊杂汤。现在不行了，腰受不了，眼睛也受不了。 headlights 打出去的光柱里开始有重影，她知道自己快到极限了。

但还有七十公里才到下一个服务区。

她揉了揉眼睛，从扶手箱里摸出一罐红牛，用牙咬开拉环。甜腻腻的味道冲进口腔，她皱了皱眉，还是喝了一大口。这东西不管用，但她需要那个仪式——拉开罐子、喝下去、告诉自己"我还能撑"。

许林动了动，醒了。

她迷迷糊糊地坐直，用手背擦了擦嘴角，"到哪了？"

"不知道。"周茹说，"还在路上。"

许林沉默了一会儿，像是在消化这个回答。然后她看向窗外，外面什么都没有，只有车灯照亮的柏油路面，和路面边缘偶尔闪过的反光条。

"你经常这样开夜车？"她问。

"经常。"

"不困吗？"

周茹没回答。她想说的是"困，但睡不着"，但话到嘴边又咽回去了。跟这个才认识不到三小时的女孩说这些干什么。

"你包里有什么吃的吗？"周茹换了个话题，"我饿了。"

许林愣了一下，然后弯腰去够脚边的双肩包。包带断了，她用那根红塑料绳系着，解开的时候有点费劲。周茹听见塑料绳摩擦的窸窣声，在安静的车厢里显得格外响。

"只有这个。"许林从包里掏出一包饼干，超市里最便宜的那种夹心饼干，包装已经有点皱了。

周茹伸手接过来，撕开包装，先递还给许林一片，自己拿了一片。饼干有点受潮，软塌塌的，甜得发腻。她慢慢嚼着，等那点糖分渗进血液里。

"你也饿了？"周茹问。

"嗯。"许林咬了一小口，"从昨天早上到现在，没吃东西。"

周茹看了她一眼。昨天早上到现在，二十四小时。这个女孩就靠着一包饼干，在高速服务区拦车。

"你去库尔勒干什么？"周茹问，"那边有熟人？"

许林嚼饼干的动作停了一下。"没有。"

"那去干什么。"

"找人。"许林说，然后像是怕周茹继续问，又补了一句，"我妈以前的朋友。"

周茹没追问。每个人都有自己的目的地，每个人都有自己的秘密。她跑这条路十二年，载过各种各样的人，有的说完目的地就一路沉默，有的会说上一大段故事然后哭着下车。她学会了不听，也不问。

但许林自己说了下去。

"我妈死前，给我留了一个地址。"她的声音很轻，像是从很远的地方飘过来，"库尔勒，建国路，十七号。她说，实在不行了，就去找这个人。"

"什么人？"

"不知道。"许林摇头，"她没说是谁。"

周茹沉默了。导航屏幕上的蓝色路线在黑暗中延伸，像是一条没有尽头的河。她想起自己也有过那样的地址，写在一张纸条上，塞在钱包的最底层。那是她儿子出事前一个月给她的，说是在新疆认识的一个朋友，让她要是路过那边就去找他喝酒。

她从来没去过。现在那个纸条还在钱包里，字迹已经模糊了。

"你确定那个人还在？"周茹问。

"不确定。"许林说，"但总得去看看。"

周茹没说话。她想起许林说的，她妈三年前死的。一个地址留了三年，现在才去找，中间这些年她在干什么？

但她没问。问了又怎样呢。答案无非是那些老套的故事——上学、打工、被骗、逃出来。她听过太多遍了。

车又开了一段，周茹的眼睛越来越重。她知道自己必须停下来了，不然就是对自己和车上这条人命不负责任。她看了眼路牌，下一个服务区还有十二公里。她咬咬牙，踩深了一点油门。

那十二公里像是一百二十公里那么长。

等她终于把车开进服务区的停车场时，她的手已经在抖了。她熄火，拉上手刹，坐在那里没动，等着那阵虚汗过去。

"到了？"许林问。

"到了。"周茹说，"睡觉。"

她下车，从车后厢拖出自己的睡袋，铺在驾驶座后面的空位上。那里本来是用来堆杂物的，她清出了一小块地方，刚好能躺下一个人。她钻进去，把拉链拉到下巴，背对着许林。

"你也睡吧。"她说，"明天早上八点走。"

许林没说话。周茹听见她在座位上窸窸窣窣地动了动，把那个破双肩包垫在头后面，蜷缩起来。

车里安静了。

周茹闭着眼睛，却睡不着。她的身体累到了极点，脑子却停不下来。她想起许林说的那个地址，建国路十七号。她在脑子里过了一遍库尔勒的地图，好像有印象，又好像没有。她只去过库尔勒两次，都是卸了货就走，没进过城。

也许明天她可以绕道去看看。反正不顺路，但也就是多绕半个小时的事。

她这么想着，终于慢慢沉下去。在失去意识前的最后一刻，她听见许林在身后轻轻地说了一句什么。太轻了，她没听清。也许是"谢谢"，也许是"对不起"，也许只是无意义的梦呓。

她没力气回头问了。

黑暗彻底吞没了她。
