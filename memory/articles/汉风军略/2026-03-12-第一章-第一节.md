# 第一章·初醒

## 第一节·我是谁

头痛欲裂。

顾衍睁开眼时，首先映入眼帘的是一片昏黄的烛光。那光线柔和而温暖，带着某种古老的质感——不是LED的冷白，也不是白炽灯的炽烈，而是真正的火焰在纱罩后面跳动，在墙壁上投下摇曳的阴影。

他愣了足足三秒。

三秒前，他还是三十一岁的科学院院士，国家重点实验室主任，刚刚完成了一个跨学科的重大突破。庆功宴上，他多喝了几杯，然后……

然后就在这里了。

顾衍缓缓撑起身体，手掌触碰到身下的床榻。不是床垫，而是某种硬实的木架，上面铺着柔软的织物。他低头看去，身上盖着一床锦被，纹样繁复，色彩艳丽——绝对不是现代的审美。

"公子醒了？"

一个声音从旁边传来。顾衍转头，看到一个少女正跪坐在榻边，约莫十五六岁，穿着一身淡青色的襦裙，发髻简单，面容清秀。她见顾衍望来，连忙低下头，声音里带着小心翼翼的恭敬："奴婢伺候公子起身。"

顾衍没有说话。

他的大脑正在高速运转。

公子。奴婢。这身装束。这间屋子。那盏油灯。

穿越。

这个念头浮上来的时候，他出奇地平静。作为历史学家，他对"穿越"这个概念并不陌生——当然，是作为研究对象，而非亲身经历。但作为一个科学家，他更习惯用证据说话。

"现在是什么年份？"他问。

少女愣了一下，似乎没想到公子会问这个问题。但她很快低下头："回公子，是元光六年。"

元光六年。

顾衍闭上眼睛。

汉武帝刘彻，元光六年。公元前129年。距马邑之谋已经过去两年，汉匈之间的小规模冲突不断，但大规模战争还未爆发。卫青刚刚崭露头角，霍去病还是个孩子。

历史的车轮，正处在一个关键的节点上。

"我……"顾衍顿了顿，"我是谁？"

这次少女真的愣住了，抬起头，眼中闪过一丝惊慌："公子？您……您怎么了？"

"回答我。"顾衍的声音平静，却带着不容置疑的威严——那是多年科研管理和项目统筹培养出来的气场。

少女瑟缩了一下，连忙低下头："公子是……是平陵侯府的三公子，姓顾名衍，表字子渊。"

平陵侯。

顾衍在脑海中搜索这个封号。西汉列侯，平陵侯……苏建？不对，苏建是因功封侯，时间不对。再往前想……

开国功臣后裔。

西汉初年，刘邦分封功臣，其中有一支……顾姓？

他忽然想起，西汉确实有一位顾姓列侯，虽然史书记载不多，但确实是开国元勋之后，封地在平陵。只不过这一支后代不显，到武帝时期已经式微，只是靠着爵位俸禄维持体面。

"我是庶子？"顾衍问。

少女的头低得更深了："是……"

顾衍点点头。

这就说得通了。列侯庶子，有爵位的名分，却无继承之权。有钱，有仆从，有宅邸，但在朝堂上没有实职，在军中没有地位。一个尴尬的身份——高不成，低不就。

但对他来说，这反而是最好的开局。

太高，会被盯上，一举一动都在权力中心的眼皮底下。太低，又要从零开始，耗费大量时间爬升。这种"有资源但无实权"的状态，正好给了他操作的空间。

"我……昏迷了多久？"顾衍又问。

"回公子，三天了。"少女的声音带着哭腔，"那日公子从马背上摔下来，撞到了头，大夫说……说可能醒不过来。奴婢……奴婢以为……"

她没有说完，但顾衍明白了。

原主骑马摔下来，撞到了头，死了。然后他来了。

"你叫什么？"顾衍问。

"奴婢叫青竹。"少女连忙答道，"是公子的贴身侍女。"

青竹。

顾衍看了她一眼。十五六岁的年纪，面容清秀，身形纤细，因为跪坐的姿势，襦裙勾勒出少女特有的青涩曲线。她低着头，露出一截白皙的颈项，在烛光下泛着柔和的光泽。

作为一个正常男人，他注意到了这些。

但作为一个科学家，他更关注的是信息。

"青竹，"顾衍缓缓开口，"我撞到头，有些事记不清了。你跟我说说，我现在的情况。"

青竹抬起头，眼中满是担忧，但还是顺从地点点头："是，公子。"

接下来的半个时辰，顾衍从青竹口中套出了大量信息。

平陵侯顾氏，开国功臣之后，封地在京畿平陵。现任平陵侯是顾衍的父亲顾崇，已经五十八岁，身体不太好。顾崇有三子，顾衍排行第三，是庶出，生母早逝。

嫡长子顾承，年三十五，是顾崇的继承人，已经在朝中任职，人缘不错。嫡次子顾昭，年二十八，掌管家族产业，精明强干。而顾衍，年二十二，游手好闲，喜欢骑马射箭，却文不成武不就，在长安城纨绔圈子里有些名气——当然，不是什么好名声。

三天前，顾衍与人赛马，从马上摔下来，撞到了头。大夫诊断为颅内有瘀血，能不能醒来全凭天意。

现在，他醒来了。

但"醒"来的，已经不是原来的顾衍了。

"公子，您……您真的没事吗？"青竹小心翼翼地问，"要不要奴婢去请大夫？"

顾衍摇摇头，慢慢坐起身。

他低头看着自己的双手。这是一双年轻的手，骨节分明，掌心有薄茧——应该是常年骑马留下的。手指修长，指甲修剪得整齐，显然原主虽然纨绔，但在生活细节上还算讲究。

"更衣。"顾衍说。

青竹连忙起身，从旁边的衣架上取过一套深色的袍服。那是汉代的男子常服，宽袍大袖，面料上乘，但纹样并不张扬——符合一个列侯庶子的身份。

顾衍站起身，张开双臂。

青竹上前，为他解开中衣的系带。她的动作轻柔而熟练，手指偶尔触碰到顾衍的皮肤，带来微凉的触感。少女身上有一股淡淡的香气，不是现代香水的浓烈，而是某种草木的清香，混合着年轻女子特有的体香。

顾衍低头看着她。

青竹的睫毛很长，在烛光下投下淡淡的阴影。她的脸颊泛着淡淡的红晕，不知是烛光映衬还是别的缘故。因为靠得近，顾衍能清晰地看到她襦裙领口露出的锁骨，纤细而精致，肌肤如雪。

他移开目光。

现在不是想这些的时候。

作为顾衍，他有一个最大的优势——他知道历史。

元光六年，汉匈战争即将全面爆发。明年，元朔元年，卫青将率兵出击匈奴，一举成名。此后数年，卫青、霍去病将接连大胜，成为汉帝国的双璧。

而他，一个列侯庶子，如果能在这个时代找到自己的位置……

"公子，好了。"青竹退后一步，轻声说。

顾衍低头看了看自己。深色的袍服穿在身上，宽袍大袖，腰系玉带，倒也有几分贵公子的气度。他走到铜镜前，第一次看到了自己现在的模样。

镜中的年轻人约莫二十出头，面容俊朗，眉目间带着几分书卷气，却又不显得文弱。因为昏迷三天，脸色有些苍白，但眼神……

顾衍看着那双眼睛。

那双眼睛里，有一种不属于这个年纪的沉静和深邃。那是经历过现代社会高等教育、科研训练、项目管理洗礼后的目光，带着审视和思索。

"公子？"青竹在身后轻声唤道。

顾衍收回目光，转身。

"备水，我要沐浴。"他说，"然后，跟我讲讲长安城最近的事。"

"是，公子。"

青竹应声退下，步履轻盈，裙摆在身后微微摆动。顾衍看着她的背影，目光在她纤细的腰肢和臀线上停留了一瞬，然后移开。

他走到窗前，推开窗。

窗外是深夜，天上有星。没有现代城市的光污染，星辰格外明亮，银河清晰可见。远处隐约可见城墙的轮廓，那是长安城的郭城，巍峨而庄严。

公元前129年。

汉武帝时代。

顾衍深吸一口气，嘴角微微上扬。

作为一个科学家，他相信机会总是留给有准备的人。而现在，他有着两千年的历史经验作为"准备"。

这个时代的游戏规则，他会慢慢学会。

而他，终将在这个时代，写下属于自己的篇章。
