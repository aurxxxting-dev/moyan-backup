# 汉风权谋·写作规则 1.0（草案）

> 针对新项目：科学院士穿越汉武帝时期
> 类型：历史穿越 + 权谋 + 后宫/言情
> 尺度：详细描写，不保守，不粗俗

---

## 一、历史穿越核心规则

### 1.1 历史红线（不可触碰）

| 原则 | 说明 |
|------|------|
| **结果不变** | 卫青、霍去病、李广等历史人物的结局不变 |
| **大势不改** | 汉匈战争走向、汉武帝晚年政策不变 |
| **死因不变** | 历史人物的死亡原因和时间不变 |
| **过程可变** | 只改变"如何达成"，不改变"达成什么" |

### 1.2 "蝴蝶翅膀"边界

**男主可以影响的小事**：
- 某场具体战役的战术细节（最终仍是大胜）
- 某位历史人物的某个决策理由（结果相同）
- 某项技术/作物的推广时间（不改变历史大势）
- 某些人的仕途快慢（不改变最终官职/结局）

**男主不能改变的大事**：
- 巫蛊之祸的发生（但可改变具体过程）
- 太子刘据之死（但可改变死亡方式）
- 卫青霍去病的功业与结局
- 汉武帝晚年的轮台诏

### 1.3 现代知识使用原则

| 类别 | 可用程度 | 示例 |
|------|----------|------|
| **农业技术** | 高 | 改进灌溉、育种、肥料 |
| **医学知识** | 中高 | 基础卫生、外伤处理、简单手术 |
| **军工技术** | 中 | 改良炼铁、标准化生产、基础火药（谨慎） |
| **政治理论** | 中 | 改良版郡县制、科举雏形（包装为"古法"） |
| **现代科学** | 低 | 不直接搬牛顿定律，用"观察总结"包装 |

## 二、权谋官场规则

### 2.1 升迁节奏

| 阶段 | 身份 | 时间跨度 | 核心矛盾 |
|------|------|----------|----------|
| **蛰伏期** | 平民/小吏 | 1-2年 | 生存、建立人脉、展示小才能 |
| **崭露头角** | 县令/郎官 | 2-3年 | 解决具体问题，进入权贵视野 |
| **入朝为官** | 二千石/近臣 | 3-5年 | 朝堂站队、卷入权力斗争 |
| **权臣之路** | 列侯/大将军 | 5-10年 | 与各方势力博弈，辅佐/对抗外戚 |

### 2.2 政治斗争原则

- **没有永远的盟友**，只有永远的利益
- **信息就是权力**，掌握消息网络才能生存
- **借势而为**，不直接对抗，借力打力
- **留后路**，任何决策都要考虑最坏情况
- **藏拙露巧**，该低调时低调，该表现时一击必中

### 2.3 与历史人物互动

| 人物 | 关系定位 | 互动方式 |
|------|----------|----------|
| **汉武帝** | 亦君亦友亦敌 | 展现才能，保持敬畏，适时进谏，不触逆鳞 |
| **卫青** | 盟友/兄长 | 敬重，不抢功，提供后勤保障/战术建议 |
| **霍去病** | 欣赏/竞争 | 欣赏其才能，保持距离（霍去病短命） |
| **李广** | 同情/协助 | 帮助其解决困境（但不改变最终结局） |
| **东方朔** | 同僚/知己 | 朝堂盟友，互通消息 |
| **张汤** | 对手/博弈 | 政治对手，必要时合作 |
| **窦婴/田蚡** | 站队考验 | 谨慎选择，不卷入必死之局 |

## 三、女性角色与情感规则

### 3.1 女性角色类型（各有特色，避免脸谱化）

| 类型 | 身份定位 | 性格特点 | 与男主关系 |
|------|----------|----------|------------|
| **女主型** | 待定（需讨论） | 待定 | 最终伴侣 |
| **才女型** | 世家小姐/歌姬 | 琴棋书画，聪慧过人，有政治敏锐度 | 红颜知己，智囊 |
| **侠女型** | 游侠/将门之女 | 武艺高强，豪爽直率，不拘小节 | 战友，保护者 |
| **胡女型** | 匈奴/西域公主 | 野性奔放，骑射精湛，文化冲突 | 异国情缘，文化交流桥梁 |
| **侍女型** | 贴身侍女 | 温柔体贴，忠心耿耿，观察细腻 | 生活照料者，情感寄托 |
| **人妻型** | 寡妇/他人之妻 | 成熟妩媚，风情万种，有故事 | 禁忌之恋（需谨慎处理） |
| **公主型** | 皇室公主 | 高贵骄傲，政治棋子，渴望自由 | 政治联姻可能，互相利用 |

### 3.2 外貌身材描写标准

**尺度原则**：详细、具体、有美感，不避讳，不粗俗

**描写维度**：
- **面容**：脸型、眉眼、鼻唇、肌肤质感、神态
- **身材**：身高、体态、曲线、比例、肌肤光泽
- **举止**：走路姿态、坐姿、手势、眼神流转
- **服饰**：穿着、面料、露肤度、随动作的变化
- **气味**：体香、脂粉、自然气息

**描写示例**（待细化）：
```
✓ 她的腰肢纤细如柳，行走时裙摆随胯轻摆，勾勒出腰臀的柔美曲线
✓ 肌肤胜雪，颈项修长，锁骨分明，在烛光下泛着温润的光泽
✗ 她胸很大，屁股很翘，看起来很欲
```

### 3.3 情感发展节奏

- **不急于确定关系**：红颜知己众多，但最终只有一个女主
- **各有故事线**：每位女性有自己的目标、困境、成长
- **感情有得有失**：不是所有人都happy ending，有人牺牲，有人离开
- **情欲与情感分离**：有肉体关系不代表有情感归宿

### 3.4 禁忌红线

❌ **绝对不写**：
- 未成年角色情色描写
- 强迫/非自愿场景
- 过于露骨的器官描写
- 乱伦（直系亲属）

⚠️ **谨慎处理**：
- 人妻/寡妇关系（需有其合理性，不道德审判）
- 权力不对等关系（主仆、君臣）
- 多角关系的嫉妒与冲突

## 四、人物塑造规则

### 4.1 男主塑造（避免龙傲天）

| 维度 | 要求 |
|------|------|
| **才能** | 理科工科扎实，但古代实践有限制；文科知识丰富，但需要包装 |
| **弱点** | 不通武艺，需要保护；不懂古代人情世故，会犯错 |
| **成长** | 从书呆子到政治家，从理想主义到现实妥协 |
| **道德** | 有底线，但为了生存会灰度操作 |

### 4.2 配角立体化原则

- **每个配角有自己的目标**：不只是为了男主存在
- **配角有自己的困境**：家庭、仕途、情感、理想
- **配角会成长或堕落**：不是静态的
- **配角有缺陷**：好人也有自私时刻，坏人也有苦衷

### 4.3 反派塑造

- **不是纯粹的恶**：有自己的逻辑和理由
- **与男主有共同点**：让反派成为"另一条路的男主"
- **有威胁感**：不只是送经验的，是真能威胁到男主的

## 五、文风与节奏

### 5.1 历史感营造

- **适当使用汉代词汇**：但不晦涩，不影响阅读
- **描写汉代生活细节**：饮食、服饰、建筑、礼仪
- **引用或化用汉代文献**：增加真实感

### 5.2 爽点与铺垫平衡

| 内容 | 比例 | 说明 |
|------|------|------|
| **权谋智斗** | 40% | 主线，解决问题、升迁、博弈 |
| **情感戏份** | 30% | 与各路女性的相识、相处、发展 |
| **历史事件** | 20% | 参与/见证历史大事件 |
| **日常细节** | 10% | 汉代生活、人际关系维护 |

### 5.3 章节节奏

- **每章有具体事件**：不空泛叙述时间流逝
- **每3章一个小高潮**：解决问题或关系进展
- **每10章一个中高潮**：权力变动或重大事件
- **每30章一个大高潮**：历史节点或情感抉择
