# 项目档案：砺锋录

## 基础设定

**类型**：武侠 / 群像 / 朝堂江湖
**核心金手指**：主角沈默能观摩数招即领悟武学精髓，融会贯通各派绝学
**主题**：强者不是用来杀人的，是用来点灯传火的

## 世界观

**时代**：架空，类似明中期的「大胤」朝
**背景**：北境战事吃紧，江湖武力式微，皇帝组建特殊机构

**核心机构**：
- **砺锋阁**：表面选拔人才，实则磨剑育人
- **龙鳞卫**：三百死士，深入敌后的精锐小队

## 核心人物

**沈默**（主角）
- 天下第一，后被招安
- 人如其名，话少，一开口必中要害
- 冷淡的教书先生气质，没有天下第一的架子
- 金手指用来育人，而非杀人

## 叙事结构

**第一卷：殿前比武**
- 时长：七天
- 每天三场比武
- 每场有人死、有人悟、有人被点破瓶颈
- 插叙参赛者背景与执念

**沈默的角色**：
- 不是参赛者，是规则之上的观察者
- 调停死斗、点拨参赛者、当众指点过招
- 用特殊能力帮助对手突破，而非击败他们

## 写作规则

- 不预设结局，一步步来
- 每场比武都是人物小传
- 打戏服务人物，不炫技
- 沈默的介入时机要准，话要少，力要透
