# 项目档案：砺锋录 · 第二卷

## 卷名：玉门沙

## 核心主题
**人物弧光为主，五感氛围为辅**

## 三条主线

### 1. 石头铸刀
- 从"持别人的刀"到"铸自己的刀"
- 西域寻矿，遇胡商、匠人、江湖人
- "止杀"是借的，"无名"是师父的，他要找到自己的

### 2. 柳白归冢
- 剑缠白布三个月，该回剑冢面对师父
- 无情剑 vs 有鞘剑的冲突
- 与剑冢决裂或继承，二选一

### 3. 沈默寻无名
- 追查三十年雪原往事
- 遇见无名的旧部、敌人、或……无名本人？
- 揭开"止杀"刀真正的来历

## 环境氛围

**玉门关 vs 雁门关**
- 雁门：雪、白、冷、肃杀
- 玉门：沙、黄、热、苍凉

**五感关键词**
- 视觉：黄沙、烈日、驼队、古城废墟、星空
- 听觉：风沙呼啸、驼铃、胡琴、马蹄踏沙
- 嗅觉：汗味、马粪、烤羊肉、铁锈、尘土
- 触觉：沙粒的粗粝、烈日的灼烧、夜风的寒凉
- 味觉：烈酒的辛辣、羊奶的膻、干粮的沙碜

## 新人物（待定）
- 胡商首领：知道西域矿产，也知道的太多秘密
- 铸刀匠人：隐居沙漠的怪人， criteria 奇怪
- 剑冢长老：柳白的师叔或师兄，守旧派
- 无名旧部：雪原三十年的幸存者，活着的 ghosts

## 卷末目标
- 石头铸成自己的刀，命名
- 柳白与剑冢决裂，创立"有鞘剑"
- 沈默找到无名，或确认无名已死，放下七年执念
