# 言情 1.0 写作规则

> 与 aur 共同确立的写作标准，适用于言情类小说创作

---

## 一、结构规则

| 项目 | 具体标准 |
|------|----------|
| **章节构成** | 一章 = 多个小节（2-4个小节） |
| **小节字数** | 每小节 3000-4000 字（根据节奏可调整） |
| **章节总字数** | 6000-12000 字（视内容重要性） |
| **小节划分** | 每个小节聚焦一个场景、一个情感波动或一个细节 |

---

## 二、节奏规则

| 项目 | 具体做法 |
|------|----------|
| **情感铺垫** | 重大转变分 2-3 章铺垫，不急于推进 |
| **发酵时间** | 暧昧期、犹豫期要充分描写，不直接给结果 |
| **留白** | 利用读者想象力，不说透，给回味空间 |
| **停顿** | 关键场景后要有"余韵"描写，不急于转场 |

---

## 三、人物塑造规则

| 项目 | 具体做法 |
|------|----------|
| **内心独白** | 多写"她在想什么"，展现犹豫、挣扎、期待 |
| **细节暗示** | 通过小动作（绞衣角、摩挲物件）暗示心理 |
| **反复犹豫** | 人物做决定前要有反复，不干脆利落 |
| **成长曲线** | 每个人物的心理变化要有层次，不跳跃 |
| **对比反差** | 利用人物之间的反差制造张力 |

---

## 四、描写规则（五感+氛围）

| 项目 | 具体做法 |
|------|----------|
| **视觉** | 光影、颜色、轮廓、细节特写 |
| **听觉** | 环境音（风声、雨声、脚步声）衬托情绪 |
| **触觉** | 温度（手心的温度、雨水的凉意） |
| **嗅觉** | 气味（檀香、桂花香、雨后的泥土味） |
| **味觉** | 食物的口感、味道与情绪关联 |
| **氛围** | 环境描写与人物心境呼应 |

---

## 五、言情向特殊规则

| 项目 | 具体做法 |
|------|----------|
| **暧昧期** | 表白前要充分的试探、犹豫、暗示 |
| **感情确认** | 不急于确定关系，让感情自然发酵 |
| **多角关系** | 每个角色的感情都要被尊重，不厚此薄彼 |
| **细腻互动** | 重视日常小细节（递东西、眼神交汇、肢体接触） |
| **心理层次** | 写"喜欢"的不同层次（好感→依赖→爱恋→深爱） |

---

## 六、对话规则

| 项目 | 具体做法 |
|------|----------|
| **潜台词** | 话里有话，不说直白 |
| **停顿** | 对话中的沉默、欲言又止 |
| **打断** | 话说到一半被打断，制造张力 |
| **动作伴随** | 说话时伴随小动作（低头、绞手、抬眼） |
