# 基础写作规则文档 1.0

> 通用写作标准模板，适用于各类小说创作
> 从实践中提炼，持续迭代进化

---

## 一、结构规则

| 项目 | 具体标准 |
|------|----------|
| **章节构成** | 一章 = 多个小节（2-4个小节） |
| **小节字数** | 每小节 3000-4000 字（根据节奏可调整） |
| **章节总字数** | 6000-12000 字（视内容重要性） |
| **小节划分** | 每个小节聚焦一个场景、一个情节点或一个细节 |

---

## 二、节奏规则

| 项目 | 具体做法 |
|------|----------|
| **铺垫充分** | 重大转变分 2-3 章铺垫，不急于推进 |
| **发酵时间** | 关键决策、情感变化要充分描写，不直接给结果 |
| **留白** | 利用读者想象力，不说透，给回味空间 |
| **停顿** | 关键场景后要有"余韵"描写，不急于转场 |

---

## 三、人物塑造规则

| 项目 | 具体做法 |
|------|----------|
| **内心独白** | 多写人物在想什么，展现犹豫、挣扎、动机 |
| **细节暗示** | 通过小动作（绞衣角、摩挲物件）暗示心理 |
| **反复犹豫** | 人物做决定前要有反复，不干脆利落 |
| **成长曲线** | 人物的心理/能力变化要有层次，不跳跃 |
| **对比反差** | 利用人物之间的反差制造张力 |

---

## 四、描写规则（五感+氛围）

| 项目 | 具体做法 |
|------|----------|
| **视觉** | 光影、颜色、轮廓、细节特写 |
| **听觉** | 环境音（风声、雨声、脚步声）衬托情绪 |
| **触觉** | 温度（手心的温度、雨水的凉意） |
| **嗅觉** | 气味（檀香、桂花香、雨后的泥土味） |
| **味觉** | 食物的口感、味道与情绪关联 |
| **氛围** | 环境描写与人物心境呼应 |

---

## 五、对话规则（2026-02-26 更新）

| 项目 | 具体做法 | 注意事项 |
|------|----------|----------|
| **说话人标识** | 对话前/后加动作或神态，明确是谁说的 | ❌ 避免长段对话不标明说话人 |
| **潜台词** | 话里有话，不说直白 | 根据场景调整 |
| **停顿** | 对话中的沉默、欲言又止 | 用动作或环境代替省略号 |
| **打断** | 话说到一半被打断，制造张力 | 自然流畅 |
| **动作伴随** | 说话时伴随小动作（低头、绞手、抬眼） | 增强画面感 |
| **标点规范** | 减少省略号使用 | ❌ 避免"..."过多，用更直接的标点 |

## 六、行文注意事项

| 类别 | 规则 | 来源 |
|------|------|------|
| **对话清晰** | 每句对话都有明确的说话人标识 | aur 2026-02-26 反馈 |
| **标点规范** | 省略号控制在必要场景（犹豫、停顿、欲言又止） | aur 2026-02-26 反馈 |
| **视角稳定** | 不随意跳视角，保持 POV 一致性 | 通用原则 |
| **信息密度** | 每段要有推进，不水字数 | 通用原则 |
| **情绪递进** | 情绪变化要有层次，不突兀 | 通用原则 |
