# [项目] 穿越之我在古代搞钱 — 人物设定卡

## 主角：林默（穿越前：林墨）

### 基础信息
- **姓名**：林默（字谨言）
- **年龄**：18岁（穿越后身体年龄），心理年龄 26
- **身份**：寒门书生 → 商人 → 权商
- **穿越前职业**：现代电商运营总监

### 穿越前背景
- 某电商平台运营总监，擅长流量变现、供应链管理
- 父母早逝，靠自己打拼，性格务实但有底线
- 猝死后穿越到同名穷书生身上

### 性格特点
| 维度 | 表现 |
|------|------|
| 核心 | 务实精明，信奉"利益交换"但不唯利是图 |
| 优点 | 眼光毒辣、执行力强、重诺守信、护短 |
| 缺点 | 有时过于算计、不擅长表达情感、防备心重 |
| 成长方向 | 从"利益至上"到"情义兼顾" |

### 金手指：【万界商道系统】

**核心功能：**
1. **市场洞察** — 显示商品供需关系、价格波动（初期范围小，随等级扩大）
2. **配方解锁** — 解锁现代商品配方（肥皂、玻璃、水泥等），需消耗"商业点"
3. **人才雷达** — 识别人才潜力（忠诚度、能力值可视化）
4. **危机预警** — 感知针对自己的阴谋（冷却时间长，非万能）

**限制：**
- 不能直接变出现代物品，必须自己生产/仿制
- 商业点通过完成交易、开拓新市场获得
- 系统不直接给答案，只给数据支持

## 女主候选方案（选一个或综合）

### 方案 A：沈清秋（官家小姐型）
- **身份**：没落世家嫡女，父亲因党争被贬
- **性格**：外柔内刚，有政治智慧，懂琴棋书画
- **作用**：政治线引子，帮主角打开权贵圈子
- **感情线**：先合作后生情，两人互相试探到信任

### 方案 B：苏小小（江湖儿女型）
- **身份**：镖局大小姐，武力值高
- **性格**：豪爽直率，不拘小节，但心思细腻
- **作用**：安保+物流线，帮主角打通运输渠道
- **感情线**：从雇佣关系到生死之交，互补型

### 方案 C：顾盼儿（才女谋士型）
- **身份**：青楼清倌人，实为前朝遗孤
- **性格**：聪慧隐忍，擅长情报收集和人心把控
- **作用**：情报网+计谋担当，灰色地带操作
- **感情线**：互相利用到真心相待，张力最强

**建议：选 A 为主线女主，B 或 C 做红颜知己/重要配角**

## 核心配角（友情线）

### 1. 周铁柱（忠犬护卫/兄弟）
- **身份**：铁匠之子，被主角从 debt 中救出
- **性格**：憨厚忠诚，武力值担当，对主角死心塌地
- **作用**：打手+执行者，后期成长为军方代表
- **弧光**：从莽夫到独当一面的将军

### 2. 钱多宝（商业合伙人/亦敌亦友）
- **身份**：钱庄少东家，商业天才但好色浮夸
- **性格**：精明但不阴险，贪财但有原则，表面轻浮实则重情
- **作用**：金融领域合作，后期成最强盟友
- **关系**：从互相利用到生死兄弟，经常斗嘴

### 3. 孙先生（谋士/导师）
- **身份**：落魄举人，被主角聘为幕僚
- **性格**：老谋深算，看透世事，把主角当接班人培养
- **作用**：政治谋略、人脉引荐、关键时刻提醒
- **秘密**：年轻时也是穿越者？（可选伏笔）

## 反派阵营

### 明面反派：赵德海（地方豪强）
- **身份**：当地最大盐商，与官府勾结
- **特点**：贪婪短视，靠垄断和欺压起家
- **作用**：前期主要对手，被主角用商业手段击垮

### 暗线反派：肃王（皇权代表）
- **身份**：当朝王爷，手握实权
- **特点**：深沉隐忍，想要收编主角为己用
- **作用**：中后期政治斗争的核心

### 隐藏反派："先生"（神秘组织）
- **身份**：前朝余孽组织的领袖
- **特点**：神出鬼没，在暗中操控各方势力
- **作用**：终极boss，与女主身世相关
