# 《反派自救指南》项目快照

## 基本信息
- **类型**：穿越/系统/反派自救/演技流
- **主角**：陈渊（穿越者）→ 万魂真人（玄阴宗宗主）
- **核心设定**：反派扮演系统——只要外界认为你是反派，你就是反派
- **风格**：轻松诙谐+智斗+演技派

## 核心人物

### 陈渊（万魂真人）
- **身份**：玄阴宗宗主，原著终极大反派
- **性格**：冷静、机智、演技派，内心善良但被迫演魔头
- **金手指**：反派扮演系统——外界认知决定人设
- **当前状态**：成功救下三百童男童女，将叶辰打落悬崖获得传承，人设契合度提升至92%
- **新剧情**：三日后正道联盟召开"除魔大会"，将围剿玄阴宗
- **核心矛盾**：当魔头但不能真当，得假装当，还要当得所有人都信

### 千尸老人
- **身份**：玄阴宗长老，陈渊下属
- **特点**：忠心耿耿，标准魔门思维模式

### 叶辰（主角）
- **身份**：天道之子，原著主角
- **状态**：已被陈渊"轰杀"至断魂崖，获得天衍剑上古传承
- **后续**：必将归来找万魂真人复仇

## 当前进度
- **第一章**：血祭危机化解，救下三百童男童女
- **第二章**：七天准备，成功将叶辰打落悬崖获得传承
- **第三章**：待写——正道联盟"除魔大会"，被围剿的准备

## 关键设定

### 反派扮演系统
- 核心逻辑：外界认为你是反派，你就是反派
- 任务奖励：寿元+年数
- 失败惩罚：神魂俱灭
- 人设契合度：目前87%

### 玄阴宗
- 魔门大派，原著反派势力
- 七十二面玄阴幡（未炼成的噬魂幡）
- 宗门氛围：全员恶人 mindset

### 原著剧情走向
- 万魂真人需要"轰杀"叶辰，让他掉下悬崖获得奇遇
- 后期将面临正道围剿
- 三百万字后轰杀至渣（陈渊要改变这个结局）

## 伏笔与线索
1. **三百童男童女**：被送给山下小门派当奴仆/药人，陈渊暗中准备营救
2. **噬魂幡**：第一笔生意没做成，后续如何炼制？
3. **叶辰**：获得天衍剑传承，后期必将归来
4. **正道联盟**：三日后召开"除魔大会"，围剿玄阴宗
5. **系统8%发挥空间**：人设契合度已达92%，还有提升余地

## 写作要点
- 保持轻松诙谐的基调
- 陈渊的内心吐槽 vs 外在的魔头形象对比
- "演技"是核心看点——如何在不崩人设的前提下做好事
- 动作场面要写出"收力"的技术感
