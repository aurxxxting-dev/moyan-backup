## 核心设定

**主角：萧逸**
- 身份：穿越者，已通过系统富可敌国
- 性格：洒脱不羁，不拘泥于世俗，平等观念
- 当前状态：暗中跟随商队出使西域，护送阿璃回家
- 隐藏身份：拥有边军特殊渠道令牌，与边军校尉有交情
- 过去：三年前曾独自前往西域龟兹，住过半年

**双女主设定**

**沈清漪**：落魄世家大小姐
- 背景：沈家犯事（与宰相崔护有关），从千金沦为奴籍
- 性格：清冷孤傲→逐渐坚韧，外冷内热
- 成长：学会骑马、射箭、辨识草药、用刀、战斗
- 心理转变：第一次杀人（马贼），从大小姐蜕变为战士
- 对萧逸：从戒备→感激→愿意共同面对危险

**阿史那·璃（阿璃）**：边塞胡女
- 背景：突厥阿史那部贵族血脉，部落公主，被父亲卖给胡商
- 性格：倔强野性，草原生存本能强
- 技能：骑射、辨识方向、草原草药、突厥语
- 对沈清漪：从陌路人→朋友→愿意为对方守夜
- 目标：回到阿史那部，面对叔叔们的权力斗争

- 提前埋伏边军弩手在沙丘后

**当前处境**:
- 已穿越白龙堆，抵达阿史那部外围绿洲
- 明天将进入阿史那部领地
- 双女主决定用账本证据制衡鹰首领

## 第四章新增关键信息

**萧逸的产业曝光**:
- 盐场：低调运营，成本比官盐低一半，质量更好
- 这次走商货物：私盐（三十车）
- 总投入超过五千贯

**萧逸的人脉网络**:
- 边军校尉：凉州的朋友，派兵在商队路线巡逻
- 老骆：五十多岁的向导，三年前被萧逸救过命
- 石头：招募的十二名退役府兵护卫

**反派点名**:
- 崔护：权倾朝野的宰相，害得沈家家破人亡的仇人
- 黑风寨：马贼势力，凌晨夜袭商队

**当前处境**:
- 已出玉门关，进入白龙堆（西域戈壁）
- 商队损失：三名护卫战死，五人受伤
- 萧逸在暗处跟随，明面上阿璃是领队（部落公主身份），沈清漪扮作侍女（实际账房）

- 制定计划：用账本证据制衡鹰首领

**萧逸**：
- 提前布局，埋伏边军弩手逼退狼表哥
- 与阿璃、沈清漪深夜对话，表达改变世界的心愿
- 曝光更多背景：想要帮助女子证明价值

## 写作要求

**节奏原则**：
- 小节篇幅均衡，不要为了分主题而硬分
- 日常细节铺垫人物变化
- 没有轰轰烈烈，只有温暖瞬间

**人物互动**：
- 双女主之间要有化学反应
- 不要太顺利，要有波折
- 外部冲突时，女主要参与，不能旁观

**言情方向**：
- 心态渐变，不是一见钟情
- 日常细节一点点融化彼此
- 最终走到一起
