## 核心设定

**主角：萧逸**
- 身份：穿越者，已通过系统富可敌国
- 性格：洒脱不羁，不拘泥于世俗，平等观念
- 日常：与下人同桌吃饭，随和亲近

**双女主设定**

**沈清漪**：落魄世家大小姐
- 背景：沈家犯事，从千金沦为奴籍，已换过三四个主家
- 性格：清冷孤傲，外冷内热，有强烈的自尊心
- 对主角初始态度：不信任，但承认他"没有用那种眼神看我"
- 技能：会写字、会算账、不会做饭

**阿史那·璃（阿璃）**：边塞胡女
- 背景：突厥阿史那部贵族血脉，被父亲卖给胡商
- 性格：倔强野性，充满敌意，不信任"随意的人"
- 对主角初始态度：明确表达"不喜欢他"
- 特征：琥珀色瞳孔，手臂有图腾纹身，会驯兽、会跳舞

## 写作要求

**节奏原则**：
- 小节篇幅均衡，不要为了分主题而硬分
- 日常细节铺垫人物变化
- 没有轰轰烈烈，只有温暖瞬间

**人物互动**：
- 双女主之间要有化学反应
- 不要太顺利，要有波折
- 外部冲突时，女主要参与，不能旁观

**言情方向**：
- 心态渐变，不是一见钟情
- 日常细节一点点融化彼此
- 最终走到一起
