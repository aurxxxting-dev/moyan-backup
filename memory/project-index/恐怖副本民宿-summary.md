# 《我在恐怖副本里开民宿》项目快照

## 基本信息
- **类型**：恐怖/副本/生活流/反向操作/爽文
- **核心爽点**：别人在恐怖副本里拼命逃生，主角反向操作直接住下，和诡异生物互利互惠
- **风格**：轻松幽默中带点恐怖氛围，生活流为主

## 世界观设定

### 背景
全球被诡异副本入侵，随机降临在世界各地。普通人被强制拉入副本，必须完成逃生任务才能存活。

### 副本规则
- 副本有难度等级（F-SSS）
- 每个副本有隐藏规则，违反即死
- 通关奖励：生存点数、装备、技能
- 死亡=真正死亡

### 主角的特殊操作
- 发现某个F级副本「红嫁衣」异常安全（只要你不主动惹鬼）
- 直接住下来，把鬼新娘当室友
- 开始经营"副本民宿"，收取高额费用让其他玩家来"安全通关"

## 主角设定
- **姓名**：陈默
- **性别**：男
- **年龄**：28岁
- **背景**：前房产中介，失业后打零工为生。看过上百套凶宅，对诡异氛围免疫。擅长察言观色、讨价还价，脸皮厚度堪比城墙。
- **性格**：胆大心细、冷静观察、脸皮厚、有商业头脑、黑色幽默
- **能力**：暂无超能力，靠智慧和嘴炮生存

## 第一副本「红嫁衣」设定

### 副本背景
一座荒废的古代宅院，传说中有个新娘在新婚之夜被新郎抛弃，穿上嫁衣上吊自杀。

### 副本规则（表面）
1. 存活到天亮
2. 不要进入新娘的房间
3. 听到哭声不要回应

### 副本真相（陈默发现）
- 鬼新娘红袖其实很寂寞，只是想找人聊天
- 她死的时候是冬天，所以一直"感冒"流鼻涕
- 给她带礼物（红糖水、胭脂、新衣服）她会保护你
- 她讨厌的是负心汉，对好人很友善

### 主要NPC
- **红袖（鬼新娘）**：红衣女鬼，外表恐怖但内心少女，喜欢漂亮东西，容易被小恩小惠收买
- **断头管家**：待出场
