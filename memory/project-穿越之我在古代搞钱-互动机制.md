# [决策] 互动小说模式设计

## 核心理念

**"可控的分支叙事"** —— 像《底特律：变人》或《奇异人生》，选择影响细节和结局，但主线脉络不变。

## 互动机制

### 触发时机
在以下场景给出选项：
1. **人物关系节点** — 信任/背叛、合作/竞争
2. **商业决策节点** — 激进/稳健、独行/合伙
3. **情感选择节点** — 女主互动、兄弟情义考验
4. **价值观节点** — 利益/道义、自保/冒险

### 选项设计原则

| 维度 | 说明 |
|------|------|
| **数量** | 每节点 2-4 个选项 |
| **影响范围** | 短期后果（1-3章）+ 长期伏笔（后续卷） |
| **不可逆性** | 部分选择可回头，部分一旦选定改变人物关系/剧情走向 |
| **隐藏选项** | 基于主角属性（智慧/魅力/武力）解锁 |

### 状态追踪

用记忆系统记录：
```
[互动状态]
- 沈清秋好感度：X/100
- 钱多宝信任度：X/100
- 商业风格：稳健派/激进派
- 道德倾向：利己/利他/平衡
- 隐藏剧情触发：A/B/C
```

## 第一章互动示例

**场景**：林默刚穿越，面对债务和母亲病重

**选项**：
1. 【稳妥】卖掉祖宅还债，先救母亲，从零开始 → 获得"孝义"标签，初期资金紧张
2. 【冒险】借钱高利贷，赌一把系统能快速赚钱 → 获得"赌徒"标签，后期债务压力
3. 【隐忍】暂时隐瞒穿越，观察环境再做决定 → 获得"谨慎"标签，错过最佳时机但情报+1

**你的选择会影响**：
- 初期资金来源和难度
- 周铁柱对你的初始印象
- 后续解锁的剧情分支

## 叙事模式

**我的输出格式**：
```
[剧情段落]
...

[互动节点]
A. 选项一
B. 选项二  
C. 选项三

你要选哪个？（直接回复 A/B/C 或描述）
```

**你也可以**：
- 说"选A"或"选第一个"
- 描述你的选择（"我选择冒险借钱"）
- 说"看看其他选项的后果"（我会简述但不剧透）

## 保存/读档机制

- **自动保存**：每章结束自动记录状态
- **关键节点**：大章节结束时可手动"存档"
- **回溯**：可以说"回到XX节点"，但会损失部分沉浸感
