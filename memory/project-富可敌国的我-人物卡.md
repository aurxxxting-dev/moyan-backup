# 《富可敌国的我买下了两个奴隶少女》——人物卡

## 男主

**姓名**：顾言
**年龄**：22岁
**身份**：穿越者，海外归来的富商独子，富可敌国
**性格标签**：温柔、平等、现代思维、不摆架子
**核心矛盾**：用现代人的尊重态度对待奴隶，与唐朝社会格格不入

## 苏瑾（女主A/倔强型）

**姓名**：苏瑾（顾言重新赋予的名字，原姓苏）
**年龄**：17岁
**外貌**：清冷倔强，眼神带刺，即使落魄也有一股傲气
**背景**：犯官之女，父亲原是朝中五品官员，因党争获罪，家破人亡后被卖为奴
**性格**：
- 表面：凶悍、防备、攻击性
- 内心：恐惧被再次伤害，用敌意保护自己
- 触发点：对"主人"这个词极度敏感
**对顾言态度演变**：
1. 敌视（又一个想买奴隶的混蛋）
2. 困惑（他怎么不按套路出牌？）
3. 怀疑（这是欲擒故纵吗？）
4. 松动（好像...真的不一样？）
**攻略难点**：打破她的心墙，让她相信顾言不是伪善

## 柳依依（女主B/温顺型）

**姓名**：柳依依（顾言重新赋予的名字，原无名）
**年龄**：16岁
**外貌**：柔弱清秀，眼神躲闪，习惯性低头，手指总是绞在一起
**背景**：贫家女，六岁被卖，辗转三个主人家，从未被当人对待过
**性格**：
- 表面：顺从、安静、不惹事
- 内心：极度缺乏安全感，害怕被抛弃
- 触发点：任何"不要我了"的信号都会让她崩溃
**对顾言态度演变**：
1. 惶恐（新主人会是什么样的人？）
2. 不安（他为什么对我这么好？）
3. 困惑（不要我伺候？）
4. 依赖（第一次有人把我当人看）
**攻略难点**：让她相信自己值得被善待，不需要用顺从换取生存
