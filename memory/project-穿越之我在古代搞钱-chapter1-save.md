## 章节：绝境觉醒（第1-30章规划中的起始部分）

### 关键剧情节点

**1. 穿越觉醒**
- 林默（林墨）穿越到大胤朝靖州府穷书生身上
- 激活【万界商道系统】，获得制皂配方
- 状态：负债32两，母亲病重

**2. 第一桶金的选择** → **C. 强势反击**
- 面对债主王屠户，虚张声势震慑对方
- 声称结识知府幕僚，获得3日缓冲期
- 获得标签【虚张声势】

**3. 资源分配** → **C. 空手套白狼**
- 与凝香阁周娘子合作，技术入股三七分
- 零成本启动，获得5两预付分成
- 解锁【凝香阁供应商】身份

**4. 合纵连横** → **C. 合纵连横**
- 主动接触沈家，寻求政治靠山
- 初见沈清秋，以商业合作+情报交换获得认可
- 沈清秋好感度：20/100
- 获得【沈家情报网】

**5. 稳固后方** → **C. 稳扎稳打**
- 优先给母亲治病（沈清秋赠百年参）
- 修补房屋、雇人照顾母亲、分发邻里
- 获得【邻里口碑】+【稳扎稳打】特质

**6. 反击时刻** → **C. 借刀杀人**
- 利用赵德海走私盐铁的情报，借沈家散布风声
- 赵德海被迫归还货物，敌意值-40（转中立）

**7. 鸿门宴** → **C. 以退为进**（自动执行）
- 称病不赴宴，送上香皂配方一成干股
- 获得3个月发展窗口期
- 赵德海暂时转为利益相关者
