# 《穿越之我在古代搞钱》人物设定卡

## 主角：林墨

### 基础信息
| 项目 | 内容 |
|------|------|
| **原名** | 林墨（穿越前：林默，28岁，互联网产品经理） |
| **穿越身份** | 大周朝江宁府穷书生，19岁，父母双亡，家徒四壁 |
| **性格** | 表面温润如玉，内心精于算计；重情重义但底线灵活 |
| **金手指** | 【万能商业系统】— 完成商业任务获得积分，兑换现代知识/物品/技能 |

### 人物弧光
- **起点**：穷书生，被退婚，被嘲笑，走投无路
- **转折点**：激活系统，用"冰糖葫芦配方"赚到第一桶金
- **成长线**：从单纯赚钱 → 到建立商业帝国 → 到改变时代
- **内心矛盾**：现代人人平等观念 vs 古代阶级固化现实

### 核心能力（系统兑换）
1. **初级**：食品配方（冰糖葫芦、辣条、方便面...）
2. **中级**：工艺技术（玻璃、肥皂、精盐提纯...）
3. **高级**：金融知识（钱庄、票号、期货概念...）
4. **隐藏**：现代管理学（组织架构、人才培养...）

## 反派人物

### 1. 王德发（前期小BOSS）
- 江宁府恶霸，强占林墨家田产
- 被林墨用"舆论战+商业围剿"搞垮
- **作用**：展示主角手段，建立初期威信

### 2. 沈万三（中期BOSS/女主父亲）
- 沈家家主，看不上穷书生林墨
- 商业上打压林墨，逼他入赘
- **转折**：被林墨的商业才能折服，成为助力

### 3. 九千岁（后期大BOSS）
- 权倾朝野的宦官，掌控盐铁专营
- 林墨的商业帝国触碰到他的利益
- **终局**：朝堂+商战双线对决
