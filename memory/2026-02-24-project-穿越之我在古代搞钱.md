# 项目：穿越之我在古代搞钱

## 项目信息

| 项目 | 内容 |
|------|------|
| **类型** | 穿越/系统/发家致富/爽文 |
| **题材** | 古代架空（类似明清商业萌芽期） |
| **核心梗** | 现代人+系统在古代商业降维打击 |
| **主线** | 穷书生 → 商业巨贾 → 权商合一 |
| **副线** | 兄弟情 + 爱情 + 智斗权贵 |

## 核心卖点

1. **系统爽感** — 现代知识+系统辅助在古代乱杀
2. **商业细节** — 真实的古代商业逻辑，有考据感
3. **人物群像** — 不止主角强，配角也有弧光
4. **情感真实** — 友情爱情不工具人，有化学反应

## 写作目标

- **测试记忆系统**：多人物长期追踪、剧情伏笔回收、人物成长一致性
- **章节规模**：预估 100-200 章，50-100 万字
- **更新节奏**：实验性质，不承诺日更
