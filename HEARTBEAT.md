# HEARTBEAT.md - 定期检查任务

## 每小时检查项
- [ ] 检查网关状态: openclaw gateway status
- [ ] 检查是否有待处理的任务
- [ ] 如有异常，发送提醒

## 备份规则

**核心原则：即时验证，即时备份**

| 场景 | 操作 | 时机 |
|------|------|------|
| 新技能/新工具安装 | 立即验证功能 | 安装完成后立即 |
| 验证通过 | 立即备份到同步文件夹 | 验证通过后立即 |
| 计划性备份 | 每日凌晨01:00自动执行 | 固定时间 |
| 重要配置变更 | 立即手动备份 | 变更后立即 |

**流程：**
```
安装/变更 → 验证测试 → 通过后立即备份 → 记录到HEARTBEAT.md
```

## 模型切换策略

| 场景 | 模式 | 触发方式 |
|------|------|----------|
| 编程/推理/调试 | 推理模型（thinking） | 自动检测或用户说"开启推理" |
| 日常对话 | 普通模式（默认） | 默认状态 |
